# 11 艺术

**考试类型：** 其他考查科目 | **成绩呈现：** 等级 | **考核方式：** 过程性评价

---

## 考查内容说明

艺术学科包含**音乐**和**美术**两个方向，主要通过过程性评价进行考核，重点考查学生在初中阶段的艺术学习参与情况和基本艺术素养。

---

## 目录内容规划

| 文件名 | 内容说明 |
|--------|----------|
| `01_音乐考查要点.md` | 音乐基础知识、中外名曲赏析 |
| `02_美术考查要点.md` | 美术基础知识、中外名画赏析 |
| `03_艺术素养积累.md` | 艺术常识与文化背景 |

---

## 备考建议

艺术学科以过程性评价为主，建议：
1. 认真参与日常音乐和美术课程
2. 积极完成课堂作品和作业
3. 了解基本的艺术常识和经典作品
