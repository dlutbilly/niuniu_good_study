# 13 综合实践

**考试类型：** 其他考查科目 | **成绩呈现：** 等级 | **考核方式：** 过程性评价

---

## 考查内容说明

综合实践活动包含**劳动教育**和**综合实践活动**两个方面，主要通过学生日常参与情况进行过程性评价，考查学生的实践能力和综合素质。

---

## 目录内容规划

| 文件名 | 内容说明 |
|--------|----------|
| `01_劳动教育.md` | 劳动教育内容与评价要求 |
| `02_综合实践活动.md` | 研究性学习、社区服务等活动记录 |
| `03_综合素质评价.md` | 综合素质评价各维度说明 |

---

## 备考建议

综合实践以过程性评价为主，建议：
1. 积极参与学校组织的各类实践活动
2. 认真完成劳动教育相关任务
3. 做好实践活动记录与总结
4. 注意综合素质评价档案的完整性
