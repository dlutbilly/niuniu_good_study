## 第四部分：病句修改专项

### 一、六大病句类型

#### 1. 语序不当

**定义**：句子中词语的排列顺序不合理。

**常见类型：**

- **定语语序不当**：多项定语的排列顺序应为：领属性定语 → 时间地点 → 指示代词 → 数量短语 → 动词短语 → 形容词 → 名词
  - 错误：我们看了一场精彩的昨天的足球赛。
  - 正确：我们看了昨天的一场精彩的足球赛。

- **状语语序不当**：
  - 错误：他在报纸上突然看到了自己的名字。
  - 正确：他突然在报纸上看到了自己的名字。

- **关联词语序不当**：主语相同时，关联词放主语后；主语不同时，关联词放主语前。
  - 错误：虽然他年纪小，但是他的志气不小。（正确）
  - 错误：虽然年纪小，但是他的志气不小。（两个主语不同，"虽然"应在"他"前）

#### 2. 搭配不当

**定义**：句子中词语之间的搭配关系不合语法或逻辑。

**常见类型：**

- **主谓搭配不当**：
  - 错误：他的写作水平和知识都有很大的提高。
  - 正确：他的写作水平和知识都有很大的提高。（"知识"不能"提高"，应改为"丰富"）

- **动宾搭配不当**：
  - 错误：我们要发扬艰苦奋斗的精神。（"发扬"搭配"精神"正确）
  - 错误：我们要发扬艰苦奋斗的作风和优良传统。（"发扬"可搭配"传统"，但"作风"应用"保持"）

- **修饰语与中心词搭配不当**：
  - 错误：他有一双乌黑的眼睛和浓密的头发。
  - 正确：他有一双乌黑的眼睛和一头浓密的头发。

#### 3. 成分残缺或多余

**定义**：句子缺少必要的成分，或有多余的成分。

**常见类型：**

- **缺主语**：
  - 错误：通过这次活动，使我们受益匪浅。
  - 正确：通过这次活动，我们受益匪浅。（删去"使"或"通过"）

- **缺宾语**：
  - 错误：我们要努力完成老师交给我们的。
  - 正确：我们要努力完成老师交给我们的任务。

- **成分多余（语义重复）**：
  - 错误：他的家乡是湖南人。
  - 正确：他是湖南人。或：他的家乡是湖南。

#### 4. 表意不明（歧义）

**定义**：句子意思不清楚，可以有两种或多种理解。

**常见类型：**

- 错误：他请了两个学校的老师来讲课。（"两个"修饰"学校"还是"老师"？）
- 正确：他请了两所学校的老师来讲课。或：他请了两位老师来讲课。

- 错误：我看见他高兴地跑过来。（谁高兴？）
- 正确：我高兴地看见他跑过来。或：我看见他高兴地向我跑来。

#### 5. 不合逻辑

**定义**：句子的意思违反了客观规律或逻辑关系。

**常见类型：**

- **概念并列不当**：
  - 错误：他喜欢看书、看报、看小说。（"小说"属于"书"，不能并列）
  - 正确：他喜欢看书、看报。

- **否定不当（多重否定）**：
  - 错误：我们不能不否认这个事实。（三重否定=否定，意思是"我们否认这个事实"）
  - 正确：我们不能否认这个事实。

- **关联词使用不当**：
  - 错误：只有努力学习，就能取得好成绩。
  - 正确：只有努力学习，才能取得好成绩。（"只有……才……"）

#### 6. 句式杂糅

**定义**：把两种不同的句式混合在一起使用。

**常见类型：**

- 错误：他的家乡是在湖南省长沙市。
  - 正确：他的家乡是湖南省长沙市。或：他的家乡在湖南省长沙市。

- 错误：这本书的作者是由鲁迅写的。
  - 正确：这本书是由鲁迅写的。或：这本书的作者是鲁迅。

- 错误：我们班的同学大多数是独生子女占多数。
  - 正确：我们班的同学大多数是独生子女。或：我们班独生子女占多数。

### 二、标点符号使用规范

| 标点 | 正确用法 | 常见错误 |
|------|----------|----------|
| 句号（。） | 陈述句末尾 | 用于疑问句末尾 |
| 问号（？） | 疑问句、反问句末尾 | 用于陈述句末尾 |
| 感叹号（！） | 感叹句、祈使句末尾 | 过多使用 |
| 逗号（，） | 句子内部停顿 | 与句号混用 |
| 顿号（、） | 并列词语之间 | 用逗号代替 |
| 分号（；） | 并列分句之间 | 与逗号混用 |
| 冒号（：） | 提示下文，总结上文 | 用于非提示语后 |
| 引号（""） | 引用、特殊含义 | 引号内标点位置 |
| 书名号（《》） | 书名、篇名、影视名 | 用引号代替 |
| 省略号（……） | 省略内容，语气停顿 | 只用三个点 |
| 破折号（——） | 解释说明，语意转折 | 只用一个横线 |

**引号内标点规则：**
- 引用完整句子：句末标点在引号内
  - 他说："我一定要努力学习。"
- 引用词语或句子的一部分：句末标点在引号外
  - 他说的"努力学习"，让我深受启发。
