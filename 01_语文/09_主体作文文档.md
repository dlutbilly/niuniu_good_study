## 主体作文一：成长蜕变型

> **原题：** 以"我创造了新纪录"为题，写一篇作文（本文作为通用主体）
> **字数：** 约 720 字
> **适配题目：** 我创造了新纪录 / 做一粒______的种子 / 这样生活更健康 / 只要肯—— / 努力成为一束光 / 驶向那片海

---

### 原文（带标注版）

那是一个**【A·时间天气】** 阴沉的周五下午，我站在**【B·场景】** 操场跑道的起点，心里涌起一阵**【C·情绪】** 紧张。**【D·背景铺垫】** 计时器上显示的数字，是我三年来从未突破过的成绩——400米，2分10秒。

没有人知道，就在这一天，我将第一次**【E·核心挑战】** 跑进2分钟。

**【F·困难开始】** 发令枪响，一切并不顺利。前200米，我的呼吸就开始乱了节奏。我**【G·动作1】** 加速，却感到双腿像灌了铅；我**【G·动作2】** 调整步伐，结果越跑越慌。额头上渗出细密的汗珠，我咬了咬牙，心里有个声音在说：**【H·放弃的念头】** "算了，今天状态不好，下次再说吧。"

就在我几乎要放弃的时候，**【I·转折契机】** 体育老师那句"你不试，永远不知道自己的极限"突然出现在脑海中。

我深吸一口气，**【J·重新开始的动作】** 重新调整呼吸节奏，迈开步伐。这一次，我不再**【K·之前的错误方式】** 盯着计时器，而是**【L·正确的方式】** 只看着前方的跑道，把注意力放在每一步的落地上。**【M·环境/感官描写】** 风从耳边呼过，跑道在脚下向后延伸。渐渐地，呼吸变得均匀了。我感到**【N·身体/心理感受】** 双腿重新有了力量，一种从未有过的轻盈在胸腔里慢慢升腾。

没有人天生就能做到。所有看起来轻松的背后，都是无数次跌倒又爬起来的痕迹。我想，成长大概就是这样——不是某一天突然变得强大，而是在每一次"再试一次"里，悄悄积累起来的。

终于，**【O·成功时刻】** 终点线出现在视野里。**【P·环境烘托】** 下午的阳光斜斜地打在跑道上，风也停了，四周好像突然安静下来。我冲过终点，喘着粗气看向计时器——**【Q·成功结果】** 1分58秒。那一刻，我愣在原地，说不清是**【R·情绪1】** 惊喜还是**【R·情绪2】** 不敢相信，只觉得眼眶有些发热。

后来，每当**【S·类似困难】** 遇到觉得"做不到"的事，我都会想起那个**【T·回忆锚点】** 阴沉的周五下午。**【U·点题句】** 新纪录，不只是秒表上的数字，更是每一次不放弃之后，留在心里的刻度。原来，突破自己，从来不需要完美的状态，只需要那一口不肯放弃的气。

### 原文（无标注干净版，可直接背诵）

那是一个阴沉的周五下午，我站在操场跑道的起点，心里涌起一阵紧张。计时器上显示的数字，是我三年来从未突破过的成绩——400米，2分10秒。

没有人知道，就在这一天，我将第一次跑进2分钟。

发令枪响，一切并不顺利。前200米，我的呼吸就开始乱了节奏。我加速，却感到双腿像灌了铅；我调整步伐，结果越跑越慌。额头上渗出细密的汗珠，我咬了咬牙，心里有个声音在说："算了，今天状态不好，下次再说吧。"

就在我几乎要放弃的时候，体育老师那句"你不试，永远不知道自己的极限"突然出现在脑海中。

我深吸一口气，重新调整呼吸节奏，迈开步伐。这一次，我不再盯着计时器，而是只看着前方的跑道，把注意力放在每一步的落地上。风从耳边呼过，跑道在脚下向后延伸。渐渐地，呼吸变得均匀了。我感到双腿重新有了力量，一种从未有过的轻盈在胸腔里慢慢升腾。

没有人天生就能做到。所有看起来轻松的背后，都是无数次跌倒又爬起来的痕迹。我想，成长大概就是这样——不是某一天突然变得强大，而是在每一次"再试一次"里，悄悄积累起来的。

终于，终点线出现在视野里。下午的阳光斜斜地打在跑道上，风也停了，四周好像突然安静下来。我冲过终点，喘着粗气看向计时器——1分58秒。那一刻，我愣在原地，说不清是惊喜还是不敢相信，只觉得眼眶有些发热。

后来，每当遇到觉得"做不到"的事，我都会想起那个阴沉的周五下午。新纪录，不只是秒表上的数字，更是每一次不放弃之后，留在心里的刻度。原来，突破自己，从来不需要完美的状态，只需要那一口不肯放弃的气。

## 主体作文二：感悟发现型

> **原题：** 以"盘中餐"为题，写一篇作文（本文作为通用主体）
> **字数：** 约 700 字
> **适配题目：** 盘中餐 / 我生活中的一棵树 / 那个星期天 / 我读到的北京 / 一堂科学课 / 最美的眼神

---

### 原文（带标注版）

**【A·核心事物】** 那只白瓷碗，在我家厨房里出现了十几年，却从未真正被我看见过。直到那一天，我才发现，原来它盛着的，**【B·悬念句，暗示深意】** 不只是饭。

那是一个**【C·时间】** 周末的中午，**【D·触发契机】** 妈妈不在家，我第一次认真地站在厨房里，看着那碗白米饭。**【E·感官描写×3】** 米粒饱满，泛着淡淡的光泽，热气轻轻地往上升，带着一股说不清的香。我用筷子轻轻拨了拨，突然有些说不清楚的感觉——**【F·初步触动】** 这碗饭，我吃了十几年，却好像从来没有真正尝过它。

不知为何，我想起了**【G·关键人物】** 外婆。**【H·回忆细节×3句】** 每年暑假，她总是天不亮就起来，去菜市场挑最新鲜的蔬菜，回来一粒一粒地淘米，说"米要淘三遍，饭才香"。那时我嫌她啰嗦，总是催她快点。原来，**【I·事物与人物的联系】** 那碗饭和外婆之间，有着我从未察觉的联系——她每一次弯腰淘米的动作，都是在用最朴素的方式，告诉我什么叫做用心。

**【J·主题特色加分段（按需选用）】** 后来我才懂得，爱从来不是轰轰烈烈的。它是清晨厨房里第一缕炊烟，是深夜书桌旁那杯悄悄放下的热牛奶，是每一次"吃饭了"背后，那个一直在等你的人。有些话，他们从来不说出口。但那些细小的动作，早已说尽了一切。

我**【K·顿悟动作】** 低下头，重新看了看那碗饭，慢慢地舀起一勺，放进嘴里。心里有什么东西，悄悄松动了。原来，**【L·深层感悟】** 一碗饭的味道，不只是米的味道，还有种它的人的汗水，煮它的人的心意，以及那些年我没有好好珍惜的，每一顿平凡的家常饭。

从那以后，我开始**【M·行为改变】** 在吃饭前，先认真看一眼碗里的饭。**【N·首尾呼应句】** 盘中餐还是那盘中餐，但我已经不是从前那个狼吞虎咽、从不在意的我了。有些温暖，不是突然出现的，它一直都在——只是等你，慢下来，看见它。

### 原文（无标注干净版，可直接背诵）

那只白瓷碗，在我家厨房里出现了十几年，却从未真正被我看见过。直到那一天，我才发现，原来它盛着的，不只是饭。

那是一个周末的中午，妈妈不在家，我第一次认真地站在厨房里，看着那碗白米饭。米粒饱满，泛着淡淡的光泽，热气轻轻地往上升，带着一股说不清的香。我用筷子轻轻拨了拨，突然有些说不清楚的感觉——这碗饭，我吃了十几年，却好像从来没有真正尝过它。

不知为何，我想起了外婆。每年暑假，她总是天不亮就起来，去菜市场挑最新鲜的蔬菜，回来一粒一粒地淘米，说"米要淘三遍，饭才香"。那时我嫌她啰嗦，总是催她快点。原来，那碗饭和外婆之间，有着我从未察觉的联系——她每一次弯腰淘米的动作，都是在用最朴素的方式，告诉我什么叫做用心。

后来我才懂得，爱从来不是轰轰烈烈的。它是清晨厨房里第一缕炊烟，是深夜书桌旁那杯悄悄放下的热牛奶，是每一次"吃饭了"背后，那个一直在等你的人。有些话，他们从来不说出口。但那些细小的动作，早已说尽了一切。

我低下头，重新看了看那碗饭，慢慢地舀起一勺，放进嘴里。心里有什么东西，悄悄松动了。原来，一碗饭的味道，不只是米的味道，还有种它的人的汗水，煮它的人的心意，以及那些年我没有好好珍惜的，每一顿平凡的家常饭。

从那以后，我开始在吃饭前，先认真看一眼碗里的饭。盘中餐还是那盘中餐，但我已经不是从前那个狼吞虎咽、从不在意的我了。有些温暖，不是突然出现的，它一直都在——只是等你，慢下来，看见它。

## 快速改写流程（考场使用）

```
第1步（1分钟）：确认题目属于哪个框架
  → 成长蜕变型 or 感悟发现型

第2步（2分钟）：查找对应主题替换表
  → 找到需要修改的标注位置（通常只需改5-8处）

第3步（2分钟）：在草稿纸上写出替换后的关键句
  → 开头第一句 / 核心事物 / 关键人物 / 结尾点题句

第4步（35分钟）：按框架写作
  → 未列出替换的部分，直接使用原文句式，填入新内容

第5步（5分钟）：检查
  → 确认题目关键词在开头和结尾均出现
  → 确认字数在650-750字之间
  → 确认无错别字、无真实校名人名
```
