## 各科目资料说明

每个学科目录下包含以下子模块（以主要计分科目为主）：

- **考点汇总**：核心知识点梳理，覆盖全部考纲
- **题型分析**：历年真题题型归纳与解题技巧
- **易错点整理**：高频失分点与纠错方法
- **冲刺提分**：临考前快速复习要点

## 使用建议

1. 优先攻克**语文、数学、英语**三门主科（共 300 分）
2. 重点突破**道德与法治、物理**（共 160 分，开卷+实验）
3. 确保**体育与健康**过程性考核不失分（50 分）
4. 等级考查科目争取全部达到 **A 等**，满足校额到校录取条件
