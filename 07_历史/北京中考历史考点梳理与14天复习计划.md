# 北京中考历史考点梳理与14天复习计划

> **适用对象：** 北京初三历史备考学生
> **考试性质：** 等级考查科目（A/B/C/D），开卷考试，各区自主命题
> **复习周期：** 14天（含两个周末）
> **文档更新：** 2026年4月

---

## 第一部分：考情分析与复习策略

### 考试定位

2025年北京中考改革后，历史属于**考查科目**，成绩以 A、B、C、D 四个等级呈现，用于毕业并在升学中参考使用。考试形式为**开卷**，各区自主命题，70分钟。虽然不计入总分，但**A等成绩是校额到校录取的重要参考条件**，因此仍需认真备考。

### 命题趋势（根据2023-2024年北京中考真题分析）

根据北京教育考试院和北京教科院基教研中心的权威解析，近年北京中考历史命题呈现以下特点：

| 命题维度 | 具体表现 |
| :--- | :--- |
| **主干知识** | 聚焦重要史事、时代特征，以统一多民族国家发展为古代史主线 |
| **核心素养** | 考查时空观念、史料实证、历史解释、家国情怀等五大素养 |
| **题型特点** | 注重开放性、探究性、综合性，多材料题、主题学习题 |
| **北京特色** | 充分利用北京地域文化资源，关注首都发展变迁 |
| **跨学科** | 融合化学、地理等学科知识，体现跨学科主题学习 |

### 复习策略

1. **构建时间轴**：将零散的知识点串联起来，形成系统的历史发展脉络。
2. **对比与联系**：注重中外历史的横向对比，以及同一历史现象的纵向发展。
3. **关注易错点**：对于容易混淆的概念和史实，要特别注意区分和记忆（文中以 ⚠️ 标记）。
4. **结合现实**：关注时代热点，思考历史与现实的联系。
5. **开卷技巧**：开卷考试不等于不用复习，要熟悉教材结构，能快速定位知识点。

### Day 1（周一）：史前时期与夏商周

**复习目标：** 了解中国早期人类的代表，掌握夏商周的更替与西周分封制。

**核心知识点：**

| 知识点 | 关键内容 |
| :--- | :--- |
| 元谋人 | 我国境内已知最早的人类，距今约170万年，云南省 |
| 北京人 | 距今约70万~20万年，北京周口店，最早使用天然火 |
| 半坡居民 | 黄河流域，种植粟，使用磨制石器，住半地穴式房屋 |
| 河姆渡居民 | 长江流域，种植水稻，住干栏式建筑 |
| 夏朝建立 | 约公元前2070年，禹建立，我国历史上第一个国家 |
| 西周分封制 | 目的：稳定周初政治形势，巩固疆土；分封对象：王族、功臣、先代贵族 |

⚠️ **易错点：**
- 人和动物的根本区别是**会不会制造工具**，而不是会不会使用工具。
- 西周的分封制不仅分封王族，还分封**异族功臣和先代贵族**，功臣和先代贵族与周天子**并无血缘关系**。
- 我国历史上第一个奴隶制国家是**夏朝**（约公元前2070年，禹建立）。

### Day 2（周二）：春秋战国与秦汉时期

**复习目标：** 掌握百家争鸣的主要思想，理解秦始皇巩固统一的措施，了解汉武帝大一统与丝绸之路。

**核心知识点：**

| 知识点 | 关键内容 |
| :--- | :--- |
| 百家争鸣 | 儒（孔子：仁；孟子：仁政）、道（老子：无为）、法（韩非：法治）、墨（墨子：兼爱非攻） |
| 秦统一六国 | 公元前221年，秦始皇嬴政统一六国，建立我国历史上第一个统一的多民族的中央集权国家 |
| 秦始皇巩固统一 | 政治：皇帝制度、三公九卿、郡县制；经济：统一货币（圆形方孔半两钱）、度量衡；文化：统一文字（小篆）；军事：修长城 |
| 汉武帝大一统 | 政治：颁布"推恩令"；思想：罢黜百家，独尊儒术；经济：盐铁专卖；军事：反击匈奴 |
| 丝绸之路 | 张骞出使西域，路线：长安→河西走廊→西域→中亚→西亚→欧洲 |

⚠️ **易错点：**
- 老子是**道家学派**的创始人，但不是**道教**的创始人，道教兴起于东汉。
- "罢黜百家，独尊儒术"并不意味着仅仅是儒家思想得到继承，因为这一时期还吸取了法家、道家等思想。
- 公元前60年设置**西域都护**，标志着西域正式归属中央政权，这是**汉宣帝**时期，不是汉武帝时期。
- **西域**指今新疆和更西的地方，而非西藏。

### Day 3（周三）：三国两晋南北朝与隋唐

**复习目标：** 掌握北魏孝文帝改革的内容与意义，理解隋朝科举制和大运河的历史意义，了解唐朝的繁荣与开放。

**核心知识点：**

| 知识点 | 关键内容 |
| :--- | :--- |
| 北魏孝文帝改革 | 迁都洛阳；说汉语、穿汉服、改汉姓、与汉族通婚；促进了北方民族的交融 |
| 科举制 | 隋朝创立，唐朝完善（武则天创立殿试，唐玄宗时诗赋成为进士科主要考试内容） |
| 大运河 | 隋炀帝开凿，以洛阳为中心，南起余杭，北到涿郡，促进南北经济文化交流 |
| 贞观之治 | 唐太宗李世民，重视人才（魏征），轻徭薄赋，政治清明 |
| 开元盛世 | 唐玄宗前期，任用贤才（姚崇、宋璟），励精图治，唐朝进入全盛时期 |
| 唐朝对外交往 | 玄奘西行（天竺）、鉴真东渡（日本）、遣唐使 |

⚠️ **易错点：**
- 唐朝流通的货币"**开元通宝**"是**唐高祖**下令造的，意为开辟新纪元，而不是唐玄宗开元年间铸造的。
- 武则天在位期间选拔的贤才有**姚崇、宋璟**，而后来唐玄宗继续重用了姚崇。
- 东汉张衡发明的**地动仪**，是**测定地震方位**的，而不是预报地震的。

### Day 4（周四）：宋元时期

**复习目标：** 理解宋代经济重心南移的原因，掌握四大发明的发展历程，了解元朝行省制度。

**核心知识点：**

| 知识点 | 关键内容 |
| :--- | :--- |
| 北宋政治 | 重文轻武；设参知政事（副宰相）、枢密使（军权）、三司使（财权）分割宰相权力 |
| 经济重心南移 | 从唐朝中期开始，到南宋时完成，"苏湖熟，天下足" |
| 四大发明 | 造纸术（汉）→印刷术（隋唐雕版，北宋活字）→火药（唐中期记载，唐末用于军事）→指南针（北宋用于航海，南宋广泛用于航海） |
| 元朝行省制度 | 在中央设中书省，地方设行中书省（行省），是我国省级行政区的开端 |
| 元朝民族关系 | 回族形成，民族交融加强 |

⚠️ **易错点：**
- 经济重心南移**完成**的时间是**南宋**，不是唐朝。
- 四大发明的顺序：造纸术（汉）、印刷术（隋唐）、火药（唐中期）、指南针（北宋）。**西汉**出现世界上已知的最早的用麻做的纸，**东汉蔡伦**改进造纸术。
- 行省制度是**元朝**创立的，不是秦朝（秦朝是郡县制）。

### Day 5（周五）：明清时期

**复习目标：** 掌握明清君主专制强化的措施，理解郑和下西洋的意义，了解清朝对边疆的管辖与闭关锁国政策。

**核心知识点：**

| 知识点 | 关键内容 |
| :--- | :--- |
| 明朝君主专制强化 | 废丞相，权分六部；设殿阁大学士（内阁）；设锦衣卫、东厂（特务机构） |
| 清朝君主专制强化 | 雍正帝设军机处（君主专制达到顶峰）；大兴文字狱 |
| 郑和下西洋 | 1405-1433年，7次，最远到达非洲东海岸和红海沿岸，是世界航海史上的壮举 |
| 清朝边疆管辖 | 西藏：设驻藏大臣；新疆：平定大小和卓叛乱，设伊犁将军；台湾：设台湾府 |
| 闭关锁国政策 | 原因：自给自足的自然经济，担心外来势力威胁统治；影响：使中国逐渐落后于世界发展潮流 |

⚠️ **易错点：**
- 清朝乾隆平定的回部贵族**大小和卓**叛乱，他们是**新疆维吾尔族**的，不是回族。
- 清朝与以往相比，巩固和发展统一多民族国家的突出特点：一是反击外来侵略，维护国家领土主权完整；二是把平定少数民族叛乱与加强民族联系和军事行政管理相结合。

### Day 6（周六）：中国古代史综合复习与自测

**复习目标：** 回顾Day 1至Day 5的内容，整理中国古代史时间轴，完成综合测试。

**重点梳理：历代王朝更替时间轴**

```
夏（约前2070）→ 商（约前1600）→ 西周（前1046）→ 春秋战国（前770）
→ 秦（前221）→ 西汉（前202）→ 东汉（25）→ 三国（220）
→ 西晋（265）→ 东晋（317）→ 南北朝（420）→ 隋（581）
→ 唐（618）→ 五代十国（907）→ 北宋（960）→ 南宋（1127）
→ 元（1271）→ 明（1368）→ 清（1644）→ 中华民国（1912）
```

**重点专题：中国古代政治制度演变**

| 时期 | 地方制度 | 中央制度 |
| :--- | :--- | :--- |
| 西周 | 分封制 | 礼乐制度 |
| 秦朝 | 郡县制 | 三公九卿制 |
| 汉朝 | 郡国并行制 | 中朝与外朝 |
| 元朝 | 行省制度 | 中书省 |
| 明朝 | 承宣布政使司 | 废丞相，六部直属皇帝 |
| 清朝 | 省制 | 军机处（君主专制顶峰） |

### Day 7（周日）：中国近代史开端（1840-1900）

**复习目标：** 掌握晚清三大不平等条约的内容及影响，理解近代化早期探索的异同。

**核心知识点：**

| 事件 | 时间 | 条约/结果 | 影响 |
| :--- | :--- | :--- | :--- |
| 第一次鸦片战争 | 1840-1842年 | 《南京条约》 | 中国开始沦为半殖民地半封建社会 |
| 第二次鸦片战争 | 1856-1860年 | 《北京条约》等 | 半殖民地化程度加深 |
| 甲午中日战争 | 1894-1895年 | 《马关条约》 | 半殖民地化程度大大加深 |
| 八国联军侵华 | 1900-1901年 | 《辛丑条约》 | 中国完全沦为半殖民地半封建社会 |
| 洋务运动 | 19世纪60-90年代 | 失败 | 中国近代化的开端 |
| 戊戌变法 | 1898年 | 失败 | 在思想文化方面产生了深远影响 |

⚠️ **易错点：**
- 中国近代史的开端是**1840年鸦片战争**，不是洋务运动。
- 《南京条约》割让的是**香港岛**给英国，不是割整个香港（香港包括香港岛、九龙、新界）。
- **火烧圆明园**是**英法联军**（1860年），不是八国联军。但两者都曾侵占过北京。
- 八国联军侵华的主要目的是**镇压义和团运动**，根本目的是维护和扩大在华利益。
- 八国联军中的"奥"指**奥匈帝国**，不是奥地利。与清政府签订《辛丑条约》的是**十一国**，不是八国。
- 洋务运动是一次**封建统治者的自救运动**，不是资产阶级性质的改革，它只学习西方技术，没有触动封建制度。

### Day 8（周一）：新民主主义革命的开始与国共合作

**复习目标：** 掌握辛亥革命的意义，理解五四运动的地位，了解中国共产党的诞生与国共合作历程。

**核心知识点：**

| 事件 | 时间 | 关键内容 |
| :--- | :--- | :--- |
| 辛亥革命 | 1911年 | 推翻清王朝，结束两千多年封建君主专制制度，建立中华民国 |
| 新文化运动 | 1915年 | 口号：民主与科学；代表人物：陈独秀、李大钊、鲁迅、胡适 |
| 五四运动 | 1919年 | 导火线：巴黎和会中国外交失败；意义：中国新民主主义革命的开端 |
| 中共一大 | 1921年 | 宣告中国共产党成立，中心：上海（后转至嘉兴南湖） |
| 北伐战争 | 1926-1927年 | 国共第一次合作，打倒军阀，基本消灭北洋军阀主力 |
| 南昌起义 | 1927年8月1日 | 打响了武装反抗国民党反动统治的第一枪 |
| 红军长征 | 1934-1936年 | 遵义会议（党史上的转折点），1936年三军会师 |

⚠️ **易错点：**
- 辛亥革命推翻了清王朝的统治，结束了两千多年的封建君主专制制度，但**并未结束**中国半殖民地半封建社会的历史。
- 五四运动取得了**阶段性胜利**（或初步胜利），不是彻底胜利。
- 五四运动是中国新民主主义革命的开端，依据是**工人阶级**在斗争中以独立姿态登上政治舞台。
- 孙中山先生**没有亲身参与**武昌起义。

### Day 9（周二）：抗日战争与解放战争

**复习目标：** 掌握抗日战争的主要史实和胜利原因，了解解放战争的主要战役。

**核心知识点：**

| 事件 | 时间 | 关键内容 |
| :--- | :--- | :--- |
| 九一八事变 | 1931年 | 日本侵占中国东北，抗日战争开始 |
| 西安事变 | 1936年 | 张学良、杨虎城逼蒋抗日，和平解决，国共第二次合作初步形成 |
| 七七事变 | 1937年7月7日 | 全面抗战爆发 |
| 抗战胜利 | 1945年8月15日 | 日本宣布无条件投降 |
| 重庆谈判 | 1945年 | 签订《双十协定》，内战爆发 |
| 三大战役 | 1948-1949年 | 辽沈、淮海、平津战役，歼灭国民党军主力 |
| 渡江战役 | 1949年4月 | 解放南京，推翻国民党政权 |

⚠️ **易错点：**
- 抗日战争胜利的根本原因是**全民族团结抗战**，而不是某一单一因素。
- 三大战役的顺序：**辽沈战役→淮海战役→平津战役**。
- 中国拒绝在《凡尔赛合约》上签字的最主要原因是中国人民的反帝活动（五四运动）。

### Day 10（周三）：中华人民共和国的成立与巩固

**复习目标：** 理解新中国成立的历史意义，掌握抗美援朝、土地改革、三大改造的内容。

**核心知识点：**

| 事件 | 时间 | 关键内容 |
| :--- | :--- | :--- |
| 开国大典 | 1949年10月1日 | 中华人民共和国成立，结束了半殖民地半封建社会历史 |
| 抗美援朝 | 1950-1953年 | 中国人民志愿军，彭德怀指挥，保家卫国 |
| 土地改革 | 1950-1952年 | 废除封建土地所有制，农民获得土地，1952年完成 |
| 一五计划 | 1953-1957年 | 集中力量发展重工业，鞍山钢铁公司、长春第一汽车制造厂等 |
| 三大改造 | 1953-1956年 | 对农业、手工业、资本主义工商业的社会主义改造，1956年完成 |
| 中共八大 | 1956年 | 正确分析了国内主要矛盾，是探索建设社会主义道路的良好开端 |

⚠️ **易错点：**
- 结束两千多年**封建帝制**的是辛亥革命；结束两千多年**封建土地所有制**的是1952年土地改革的完成；结束**半殖民地半封建社会**历史的是开国大典（1949年）。
- 三大改造完成的标志是**1956年**，标志着社会主义制度在我国基本建立。

**复习目标：** 掌握十一届三中全会的历史意义，了解改革开放的主要内容和成就。

**核心知识点：**

| 事件 | 时间 | 关键内容 |
| :--- | :--- | :--- |
| 十一届三中全会 | 1978年 | 党史上的重大转折点；确立改革开放和社会主义现代化建设的历史新时期 |
| 农村改革 | 1978年起 | 家庭联产承包责任制，安徽凤阳小岗村率先实行 |

| 社会主义市场经济 | 1992年 | 党的十四大确立建立社会主义市场经济体制的目标 |

### Day 12（周五）：世界古代史与近代史

**复习目标：** 掌握古代文明的主要成就，理解文艺复兴和新航路开辟的意义，了解英美法资产阶级革命和工业革命。

**核心知识点：**

| 知识点 | 关键内容 |
| :--- | :--- |
| 四大文明古国 | 古埃及（尼罗河）、古巴比伦（两河流域）、古印度（印度河）、中国（黄河） |
| 雅典民主政治 | 伯里克利时期达到顶峰，成年男性公民可参与政治，但妇女、奴隶、外邦人除外 |
| 文艺复兴 | 14-17世纪，意大利兴起，核心思想：人文主义；代表人物：但丁、达·芬奇、莎士比亚 |
| 新航路开辟 | 迪亚士（好望角）→达·伽马（印度）→哥伦布（美洲）→麦哲伦（环球航行） |
| 英国资产阶级革命 | 1640年爆发，1688年光荣革命结束，1689年《权利法案》确立君主立宪制 |
| 美国独立战争 | 1775-1783年，1776年《独立宣言》，确立了共和政体 |
| 法国大革命 | 1789年，《人权宣言》，拿破仑帝国 |
| 第一次工业革命 | 18世纪60年代，英国，瓦特改良蒸汽机，进入"蒸汽时代" |
| 马克思主义诞生 | 1848年，《共产党宣言》发表，科学社会主义诞生 |

⚠️ **易错点：**
- 文艺复兴的实质是**资产阶级思想解放运动**，而不是古代希腊罗马文化的简单复兴。
- 首先直接实现西欧人开辟新航路基本目标（到达东方）的是**达·伽马**，而不是哥伦布（哥伦布到达的是美洲）。
- 英国资产阶级革命**结束**的标志是1688年的"**光荣革命**"，而不是《权利法案》的颁布（后者标志着君主立宪制的确立）。
- 第一次工业革命的发明：**哈格里夫斯**发明珍妮机（棉纺织业），**瓦特**改良蒸汽机（动力革命）。

### Day 13（周六）：世界现代史

**复习目标：** 掌握俄国改革、日本明治维新的特点，了解两次世界大战和战后格局。

**核心知识点：**

| 知识点 | 关键内容 |
| :--- | :--- |
| 俄国1861年改革 | 沙皇亚历山大二世，废除农奴制，自上而下的资产阶级性质改革，保留了大量封建残余 |
| 日本明治维新 | 1868年，向西方学习，富国强兵，自上而下的资产阶级性质改革，保留了大量封建残余 |
| 第一次世界大战 | 1914-1918年，导火线：萨拉热窝事件，根本原因：帝国主义国家政治经济发展不平衡 |
| 十月革命 | 1917年，列宁领导，建立世界上第一个社会主义国家 |
| 罗斯福新政 | 1933年，特点：国家干预经济；核心：整顿金融，颁布《全国工业复兴法》 |
| 第二次世界大战 | 1939-1945年，1939年德国闪击波兰，1941年苏德战争、太平洋战争爆发 |
| 冷战 | 1947年开始，美苏两极格局，1991年苏联解体，冷战结束 |

⚠️ **易错点：**
- 俄国1861年改革和日本明治维新都是**自上而下**的资产阶级性质的改革，都保留了大量的**封建残余**。
- 第一次世界大战的**导火线**是萨拉热窝事件，**根本原因**是帝国主义国家政治经济发展不平衡，**直接原因**是帝国主义国家之间矛盾激化。
- 罗斯福新政**没有改变**资本主义的本质，也**无法从根本上消除**经济危机。
- 二战中，**斯大林格勒战役**是二战的重要转折点，而不是莫斯科保卫战。
