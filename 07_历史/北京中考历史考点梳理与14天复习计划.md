# 北京中考历史考点梳理与14天复习计划

> **适用对象：** 北京初三历史备考学生
> **考试性质：** 等级考查科目（A/B/C/D），开卷考试，各区自主命题
> **复习周期：** 14天（含两个周末）
> **文档更新：** 2026年4月
> **知识来源：** 结合《初中历史知识点总结》（下划线处为填空考点）+ 近5年北京中考真题分析

---

## 第一部分：考情分析与复习策略

### 考试定位

2025年北京中考改革后，历史属于**考查科目**，成绩以 A、B、C、D 四个等级呈现，用于毕业并在升学中参考使用。考试形式为**开卷**，各区自主命题，70分钟。虽然不计入总分，但**A等成绩是校额到校录取的重要参考条件**，因此仍需认真备考。

### 命题趋势（根据2021-2025年北京中考真题分析）

根据北京教育考试院和北京教科院基教研中心的权威解析，近年北京中考历史命题呈现以下特点：

| 命题维度 | 具体表现 |
| :--- | :--- |
| **主干知识** | 聚焦重要史事、时代特征，以统一多民族国家发展为古代史主线 |
| **核心素养** | 考查时空观念、史料实证、历史解释、家国情怀等五大素养 |
| **题型特点** | 注重开放性、探究性、综合性，多材料题、主题学习题 |
| **北京特色** | 充分利用北京地域文化资源，关注首都发展变迁 |
| **跨学科** | 融合化学、地理等学科知识，体现跨学科主题学习 |

### 复习策略

1. **构建时间轴**：将零散的知识点串联起来，形成系统的历史发展脉络。
2. **对比与联系**：注重中外历史的横向对比，以及同一历史现象的纵向发展。
3. **关注易错点**：对于容易混淆的概念和史实，要特别注意区分和记忆（文中以 ⚠️ 标记）。
4. **结合现实**：关注时代热点，思考历史与现实的联系。
5. **开卷技巧**：开卷考试不等于不用复习，要熟悉教材结构，能快速定位知识点。

## 第三部分：每日知识考点详解

---

### Day 1（周一）：史前时期与夏商周

**复习目标：** 了解中国早期人类的代表，掌握夏商周的更替、西周分封制，理解甲骨文的历史意义。

**对应教材：** 七年级上册 第一单元、第二单元

---

#### 一、史前时期：中国境内早期人类与文明的起源

**1. 北京人（★★★高频考点）**

| 要素 | 内容 |
| :--- | :--- |
| 时间 | 距今约 **70万—20万年** |
| 地点 | 北京 **周口店** 龙骨山 |
| 特征 | 能直立行走；使用**打制石器**；会**使用天然火** |
| 意义 | 周口店北京人遗址是迄今所知世界上内涵最丰富、材料最齐全的**直立人遗址**之一；为人类起源研究提供可靠证据 |

> 📝 **填空考点：** 化石是研究人类起源的**主要依据**。

**2. 原始农耕生活**

| 居民 | 地点 | 时间 | 农作物 | 房屋 | 其他 |
| :--- | :--- | :--- | :--- | :--- | :--- |
| 河姆渡居民 | 浙江余姚（**长江**流域） | 距今约**七千年** | **水稻** | **干栏式**房屋 | 会挖掘水井；使用磨制石器 |
| 半坡居民 | 陕西西安（**黄河**流域） | 距今约**六千年** | **粟** | **半地穴式**房屋 | 制造彩陶 |

> 📝 **填空考点：** 目前，世界上最早的栽培**稻**、**粟**和**黍**均发现于中国。

**3. 炎黄传说与禅让制**

- 距今五六千年，黄帝部落和炎帝部落联合，通过**涿鹿之战**打败蚩尤部落，形成**华夏族**，后人尊称为**中华民族的人文初祖**。
- **禅让制**：将联盟首领的位子传给**贤德之人**（尧→舜→禹）。

⚠️ **易错点：**
- 人和动物的根本区别是**会不会制造工具**，而不是会不会使用工具。
- 元谋人（云南，距今约**170万年**）是我国境内已知**最早**的人类；北京人**不是**最早的。

---

#### 二、早期国家的产生和社会变革

**4. 夏商周的更替**

| 朝代 | 建立时间 | 建立者 | 都城 | 灭亡原因 |
| :--- | :--- | :--- | :--- | :--- |
| 夏朝 | 约公元前**2070**年 | **禹** | 阳城 | 夏桀暴政，商汤灭夏 |
| 商朝 | 约公元前**1600**年 | **商汤** | 亳（后迁殷） | 商纣残暴，武王伐纣 |
| 西周 | 公元前**1046**年 | **周武王** | 镐京 | 平王东迁，进入东周 |

> 📝 **填空考点：** 禹传**启**，世袭制取代**禅让制**，"公天下"变成"**家天下**"，我国由原始社会进入**奴隶社会**。

**5. 西周分封制（★★★大题考点）**

- **目的：** 巩固统治
- **内容：** 周天子根据**血缘关系**远近和**功劳大小**，把土地、平民、奴隶分给亲属、功臣等，封他们为诸侯；诸侯必须服从周天子命令，交纳贡品、镇守疆土、带兵随周天子作战。
- **意义：** 开发了边疆，加强了统治，西周成为强盛的国家；维护了西周王朝严格的**等级制度**。

**6. 商朝的文明**

- **甲骨文：** 商朝人刻写在**龟甲或兽骨**上的文字，是一种比较成熟的文字，具备了汉字的基本结构。我国有文字可考的历史，从**商朝**开始。
- **青铜器：** 迄今世界上出土的**最重**的青铜器是商朝的**司母戊鼎**。

> 📝 **填空考点：** 我国有文字可考的历史，从**商朝**开始（甲骨文）。

⚠️ **易错点：**
- 我国历史上第一个奴隶制国家是**夏朝**（约公元前2070年，禹建立）。
- 西周的分封制不仅分封王族，还分封**异族功臣和先代贵族**，功臣和先代贵族与周天子**并无血缘关系**。
- **甲骨文**是商朝文字，**金文**是铸刻在青铜器上的文字，两者都是汉字的早期形态。

### Day 2（周二）：春秋战国与秦汉时期

**复习目标：** 掌握百家争鸣的主要思想，理解秦始皇巩固统一的措施，了解汉武帝大一统与丝绸之路。

**对应教材：** 七年级上册 第三单元、第四单元

---

#### 一、春秋战国时期

**1. 春秋争霸与战国七雄**

- 春秋五霸：齐桓公、晋文公、楚庄王、秦穆公等先后称霸中原。
- 战国七雄：**齐、楚、秦、燕、赵、魏、韩**（东南西北到中间）。
- 著名战役：**桂陵之战、马陵之战、长平之战**。

**2. 商鞅变法（前356年，秦孝公时期）（★★★大题考点）**

| 领域 | 内容 |
| :--- | :--- |
| 政治 | 建立**县制**，由国君直接派官吏治理；废除旧贵族的**世袭特权**；改革户籍制度；严明法度，禁止私斗 |
| 经济 | 废**井田**，允许土地自由买卖；奖励**耕织**，生产粮食布帛多的人可免除徭役；统一**度量衡** |
| 军事 | 奖励**军功**，对有军功者授予爵位并赏赐土地 |

> 📝 **填空考点：** 商鞅变法使秦国的国力大增，提高了军队战斗力，一跃成为最强盛的诸侯国，为以后秦国**统一全国**奠定了基础。

**3. 都江堰**

- 战国时期，秦国蜀郡郡守**李冰**修筑，发挥着**防洪、灌溉、水运**等多方面的作用，使成都平原获得了"**天府之国**"的美称。

**4. 百家争鸣（★★★大题考点）**

| 学派 | 代表人物 | 主要思想 |
| :--- | :--- | :--- |
| 儒家 | **孔子**（春秋） | 核心思想"**仁**"，"仁者爱人"；主张"以德治国"；创办私学，"**有教无类**"；记录其言行的是《**论语**》 |
| 儒家 | **孟子**（战国） | 主张实行"**仁政**"；"民为贵，社稷次之，君为轻"；反对一切非正义战争 |
| 儒家 | **荀子**（战国） | 主张实行**礼治**，明确尊卑等级 |
| 道家 | **老子**（春秋） | 著有《**道德经**》；主张事物有对立面，会互相转化；主张**无为而治** |
| 道家 | **庄子**（战国） | 强调治国要顺应自然与民心；追求精神自由 |
| 墨家 | **墨子**（战国） | 主张**兼爱、非攻**、尚贤、节俭 |
| 法家 | **韩非**（战国） | 法家集大成者；强调以**法治国**，树立君主权威，建立**中央集权**制度 |

> 📝 **填空考点：** 百家争鸣促进了思想和学术的繁荣，是我国古代**第一次**思想文化发展的高峰。

---

#### 二、秦汉时期

**5. 秦统一中国（★★★大题考点）**

- 公元前**221**年，秦王**嬴政**完成统一大业，建立我国历史上第一个**统一的多民族的封建国家**——秦朝，定都**咸阳**。

**秦朝巩固统一的措施：**

| 领域 | 措施 |
| :--- | :--- |
| 政治 | 创立皇帝制度；中央设**丞相**（行政）、**太尉**（军事）、**御史大夫**（监察）；地方废分封，实行**郡县制** |
| 文化 | 统一文字，通用**小篆** |
| 经济 | 统一货币（**圆形方孔半两钱**）；统一度量衡 |
| 交通 | 统一车辆和道路宽窄；修筑贯通全国的道路；开凿**灵渠** |
| 军事 | 北击匈奴；修筑长城（西起**临洮**，东到**辽东**） |

> 📝 **填空考点：** 郡县制的实行，开创了此后我国历代王朝地方行政的**基本模式**。

**6. 汉武帝巩固大一统（★★★大题考点）**

| 领域 | 措施 |
| :--- | :--- |
| 政治 | 接受主父偃建议，颁布"**推恩令**"削弱诸侯；设立**刺史**制度，加强中央对地方的控制 |
| 思想 | 接受董仲舒建议，"**罢黜百家，独尊儒术**"；兴办**太学**（我国古代最高学府），以儒家"五经"为主要教材 |
| 军事 | 派**卫青、霍去病**北击匈奴 |
| 外交 | 派**张骞**两次出使西域，开辟丝绸之路 |
| 经济 | 将铸币权收归中央，统一铸造**五铢钱**；实行**盐铁专营**；统一调配物资、平抑物价 |

**7. 丝绸之路（★★★大题考点）**

- **陆上丝绸之路：** 长安→**河西走廊**→西域→中亚、西亚→欧洲（大秦）
- **海上丝绸之路：** 从东南沿海港口出发，最远抵达印度半岛南端和**锡兰**（今斯里兰卡）
- **意义：** 丝绸之路是古代东西方往来的大动脉，极大促进了中国同其他国家和地区的贸易与文化交流。

> 📝 **填空考点：** 公元前60年设置**西域都护**，标志着西域正式归属中央政权。

**8. 秦汉时期的科技与文化**

| 领域 | 内容 |
| :--- | :--- |
| 史学 | 司马迁著《**史记**》，我国第一部**纪传体**通史，记述从**黄帝**到**汉武帝**时期的史事；被誉为"**史家之绝唱，无韵之离骚**" |
| 造纸术 | 西汉时已懂得造纸；**东汉蔡伦**改进造纸术（"蔡侯纸"） |
| 医学 | **华佗**（东汉末年）：制成世界最早的麻醉剂"**麻沸散**"，编创"**五禽戏**" |
| 医学 | **张仲景**（东汉末年）：写成《**伤寒杂病论**》，后世称"**医圣**" |
| 宗教 | 佛教：起源于古代印度，经**丝绸之路**传入中国；道教：中国**土生土长**的宗教，**东汉末年**在民间兴起 |

⚠️ **易错点：**
- 老子是**道家学派**的创始人，但不是**道教**的创始人，道教兴起于**东汉**。
- "罢黜百家，独尊儒术"并不意味着仅仅是儒家思想得到继承，这一时期还吸取了法家、道家等思想。
- 公元前60年设置**西域都护**，标志着西域正式归属中央政权，这是**汉宣帝**时期，不是汉武帝时期。
- **西域**指今新疆和更西的地方，而非西藏。
- 《史记》是**纪传体**通史；《资治通鉴》是**编年体**通史，注意区分。

### Day 3（周三）：三国两晋南北朝与隋唐

**复习目标：** 掌握北魏孝文帝改革的内容与意义，理解隋朝科举制和大运河的历史意义，了解唐朝的繁荣与开放。

**对应教材：** 七年级上册 第五单元、七年级下册 第一单元

---

#### 一、三国两晋南北朝时期

**1. 三国鼎立**

| 战役 | 时间 | 交战双方 | 结果 | 意义 |
| :--- | :--- | :--- | :--- | :--- |
| 官渡之战 | 200年 | 曹操 vs 袁绍 | 曹操大败袁绍 | 为曹操**统一北方**奠定基础 |
| 赤壁之战 | 208年 | 曹操 vs 孙刘联军 | 孙刘联军用火攻打败曹军 | 为**三国鼎立**局面的形成奠定基础 |

| 政权 | 建立时间 | 建立者 | 都城 |
| :--- | :--- | :--- | :--- |
| 魏 | 220年 | **曹丕** | 洛阳 |
| 蜀（蜀汉） | 221年 | **刘备** | 成都 |
| 吴 | 229年 | **孙权** | 建业 |

**2. 人口南迁与江南开发**

- 西晋末年以来，大批北方人迁到江南，出现中国古代历史上**第一次**大规模人口迁徙浪潮。
- 北民南迁带来了**劳动力、先进的生产技术和生产工具**，江南经济发展迅速。
- 江南地区的开发为中国经济重心的**逐渐南移**奠定了基础。

**3. 北魏孝文帝改革（★★★大题考点）**

- **内容：** ①迁都**洛阳**；②实行汉化政策：说**汉语**、穿**汉服**、改**汉姓**、通**汉婚**、用汉制、学汉礼。
- **意义：** 促进了**民族交融**，也增强了北魏的实力。

**4. 魏晋南北朝的科技与文化**

| 领域 | 人物/内容 |
| :--- | :--- |
| 数学 | **祖冲之**（南朝）：把圆周率精确到小数点后**第七位**，领先世界近**千年** |
| 书法 | **王羲之**（东晋）：《**兰亭集序**》被称为"天下第一行书"，被誉为"**书圣**" |
| 历法 | 祖冲之创制《**大明历**》，是当时最先进的历法 |

#### 二、隋唐时期

**5. 隋朝的建立与统一**

- 581年，**杨坚**（隋文帝）建立隋朝；589年，隋文帝灭陈朝，**统一全国**，结束了长期分裂的局面。

**6. 科举制（★★★高频考点）**

- **创立：** 隋文帝初步建立通过考试选拔人才的制度；**隋炀帝**创立**进士科**，标志着科举制**正式确立**。
- **完善：** 唐朝完善，武则天创立**殿试**，唐玄宗时**诗赋**成为进士科主要考试内容。
- **意义：** 科举制是中国古代选官制度的一大变革，加强了皇帝在选官和用人上的权力，扩大了官吏选拔范围，促进了**社会阶层的流动**，推动了教育的发展。科举制成为历朝选拔官吏的主要制度，一直维持了约**1300年**。

**7. 隋朝大运河（★★★高频考点）**

- **开凿者：** 隋炀帝
- **路线：** 以**洛阳**为中心，北抵**涿郡**，南至**余杭**
- **连接：** 海河、黄河、淮河、长江、钱塘江**五大水系**
- **意义：** 加强了南北地区政治、经济和文化交流（消极：给人民带来沉重负担，加速了隋朝灭亡）

**8. 贞观之治（★★★高频考点）**

- **唐太宗**李世民统治时期，政治清明，经济发展，国力强盛，史称"贞观之治"。
- **施政措施：**
  - 政治：吸取隋亡教训，勤于政事，选贤任能，虚心纳谏（**魏征**直言；**房玄龄**善谋——**杜如晦**善断，"**房谋杜断**"）；进一步完善**三省六部制**；制定法律，减轻刑罚。
  - 经济：减轻人民劳役负担，鼓励发展农业生产。
  - 民族：实行开明的民族政策；将**文成公主**嫁给吐蕃赞普**松赞干布**，促进了吐蕃经济和社会的发展。

**9. 开元盛世**

- **唐玄宗**前期，年号"开元"，政治稳定，经济繁荣，国库充盈，唐朝进入鼎盛时期，史称"**开元盛世**"。

**10. 盛唐气象**

| 领域 | 内容 |
| :--- | :--- |
| 农业 | 农耕技术进步，农业工具改进（**曲辕犁**、**筒车**） |
| 手工业 | 纺织业以**蜀锦**冠于全国；陶瓷业以**越窑青瓷**、**邢窑白瓷**和**唐三彩**最为有名 |
| 商业 | 都城**长安**是当时中国政治、经济和文化交往的中心，也是一座**国际性的大都会** |
| 诗歌 | **李白**（"诗仙"，浪漫主义）；**杜甫**（"诗圣"，现实主义，作品有"诗史"之称）；**白居易**（平易近人，通俗易懂） |

**11. 唐朝的对外交往（★★★高频考点）**

| 人物 | 方向 | 内容 |
| :--- | :--- | :--- |
| **鉴真** | 东渡**日本** | 传播佛教，为中日文化交流做出卓越贡献 |
| **玄奘** | 西行**天竺**（今印度） | 著有《**大唐西域记**》，是研究中外交流史的珍贵文献 |
| 遣唐使 | 日本→中国 | 日本派遣唐使来华学习，促进中日文化交流 |

**12. 安史之乱**

- 755年，**安禄山**和**史思明**发动叛乱，史称"安史之乱"。
- **影响：** 对社会经济造成极大破坏；唐朝国势从此由盛转衰；逐渐形成**藩镇割据**的局面。

⚠️ **易错点：**
- 唐朝流通的货币"**开元通宝**"是**唐高祖**下令造的，意为开辟新纪元，而不是唐玄宗开元年间铸造的。
- 武则天在位期间选拔的贤才有**姚崇、宋璟**，而后来唐玄宗继续重用了姚崇。
- **科举制**创立于隋朝（隋炀帝创立进士科），不是唐朝。
- 赤壁之战是以**少胜多**的战役，孙刘联军人数远少于曹操。

### Day 4（周四）：宋元时期

**复习目标：** 理解宋代经济重心南移的原因，掌握四大发明的发展历程，了解元朝行省制度和民族交融。

**对应教材：** 七年级下册 第二单元、第三单元

---

#### 一、辽宋夏金元时期

**1. 政权并立**

| 政权 | 民族 | 建立时间 | 建立者 |
| :--- | :--- | :--- | :--- |
| 北宋 | 汉族 | 960年 | **赵匡胤**（宋太祖，陈桥兵变） |
| 辽 | 契丹族 | 916年 | 耶律阿保机 |
| 西夏 | 党项族 | 1038年 | 元昊 |
| 金 | 女真族 | 1115年 | 完颜阿骨打 |
| 元 | 蒙古族 | 1271年 | 忽必烈 |

- **澶渊之盟：** 宋辽签订，维持和平局面，宋每年向辽缴纳"**岁币**"。
- **南宋岳飞抗金：** 岳飞领导抗金斗争，取得郾城大捷等重大胜利，后被秦桧以"莫须有"罪名杀害。

**2. 宋代南方经济的发展（★★★大题考点）**

| 领域 | 内容 |
| :--- | :--- |
| 农业 | 引进**占城稻**，水稻产量跃居粮食作物首位；出现"**苏湖熟，天下足**"的谚语（反映经济重心南移完成） |
| 手工业 | 景德镇发展成为**瓷都**；广州、泉州、明州的造船业居世界领先地位 |
| 商业 | 最大商业城市是**开封**和**临安**（杭州）；出现**早市和夜市**；乡镇还有**草市** |
| 海外贸易 | 广州、泉州是闻名世界的大商港，设**市舶司**管理海外贸易 |
| 货币 | 北宋前期，四川出现世界上最早的纸币——**交子** |

> 📝 **填空考点：** 从唐朝中期开始的经济重心南移，到**南宋**时完成。"**国家根本，仰给江南**"说明经济重心的南移。

**经济重心南移的原因：**
1. 北人南迁，带来了劳动力和先进的技术、生产工具（**最主要原因**）
2. 江南地区战争相对较少，社会秩序比较安定
3. 南方优越的自然地理条件
4. 南北方人民的共同努力
5. 南方统治者大多重视发展经济

**3. 蒙古统一与元朝建立**

- 1206年，**铁木真**统一蒙古草原，建立蒙古政权，被尊为"**成吉思汗**"。
- 1271年，**忽必烈**改国号为**元**，定都**大都**（北京）。1279年，元朝完成全国统一。
- 抗元将领：**文天祥**（"人生自古谁无死，留取丹心照汗青"）。

**4. 元朝对边疆的管辖**

- 元朝建立后，由**宣政院**直接管辖**西藏**地区事务，西藏正式成为中央直接管辖下的一个地方行政区域。

**5. 元朝行省制度**

- 在中央设**中书省**，地方设**行中书省**（行省），是我国**省级行政区**的开端。

#### 二、宋元时期的科技文化

**6. 四大发明（★★★大题考点）**

| 发明 | 发明/改进时间 | 传播 | 影响 |
| :--- | :--- | :--- | :--- |
| 造纸术 | 西汉出现；**东汉蔡伦**改进 | 经丝绸之路传到欧洲 | 促进文化传播，是中国对世界文明的伟大贡献 |
| 印刷术 | **隋唐**雕版印刷；**北宋毕昇**发明活字印刷 | 13世纪传入朝鲜，后传到欧洲 | 促进文化知识传播，为欧洲文艺复兴创造了条件 |
| 火药 | **唐朝**发明；**宋代**用于军事 | 13世纪传入阿拉伯，14世纪传到欧洲 | 对欧洲火器制造和作战方式产生巨大影响，推动欧洲社会变革 |
| 指南针 | **汉代**制成"司南"；**北宋末年**用于航海 | 阿拉伯商人传到阿拉伯国家，后传到欧洲 | 大大促进了世界远洋航海技术的发展，为新航路开辟提供了条件 |

**7. 宋词与元曲**

| 类别 | 代表人物 | 风格 | 代表作 |
| :--- | :--- | :--- | :--- |
| 宋词（豪放派） | **苏轼**（北宋） | 豪迈而飘逸 | 《念奴娇·赤壁怀古》 |
| 宋词（豪放派） | **辛弃疾**（南宋） | 豪放，报国情怀 | 《破阵子》 |
| 宋词（婉约派） | **李清照**（两宋之交） | 委婉、细腻、清秀 | 《声声慢》 |
| 元曲 | **关汉卿** | 元代最优秀的戏剧家 | 《**窦娥冤**》 |

**8. 史学名著**

> 📝 **填空考点：** 司马光是北宋著名**史学家**、政治家。《**资治通鉴**》是一部**编年体**的通史巨著，记述了从**战国**到**五代**共1300多年的历史，纵贯中国16个朝代。

⚠️ **易错点：**
- 经济重心南移**完成**的时间是**南宋**，不是唐朝。
- 四大发明的顺序：造纸术（汉）、印刷术（隋唐）、火药（唐）、指南针（北宋用于航海）。**西汉**出现最早的纸，**东汉蔡伦**改进造纸术。
- 行省制度是**元朝**创立的，不是秦朝（秦朝是郡县制）。
- **交子**是北宋四川出现的世界上最早的纸币，不是元朝。
- 《史记》是**纪传体**通史；《资治通鉴》是**编年体**通史，两者都是重要史学著作，注意区分。

### Day 5（周五）：明清时期

**复习目标：** 掌握明清君主专制强化的措施，理解郑和下西洋的意义，了解清朝对边疆的管辖与闭关锁国政策。

**对应教材：** 七年级下册 第三单元、第四单元

---

#### 一、明朝时期

**1. 明朝加强君主专制（★★★大题考点）**

| 措施 | 内容 |
| :--- | :--- |
| 地方 | 取消**行中书省**，设立"三司"（承宣布政使司、提刑按察使司、都指挥使司）；分封诸子为王 |
| 中央 | **废除丞相制度**和中书省，提升六部职权，直接向皇帝负责；分散兵权，设立五军都督府 |
| 特务机构 | 设**锦衣卫**和**东厂**，监视臣民 |
| 思想控制 | 实行**八股取士**，控制思想（禁锢知识分子思想，不利于社会进步） |

> 📝 **填空考点：** 明朝废除**丞相制度**，是我国历史上延续了一千多年的丞相制度的终结。

**2. 郑和下西洋（★★★高频考点）**

- **时间：** 1405—1433年，共**7次**
- **最远到达：** **非洲东海岸**和**红海沿岸**
- **意义：** 增进了中国与亚非国家和地区的相互了解和友好往来；开创了西太平洋与印度洋之间的亚非海上交通线；是世界航海史上的**空前壮举**。

**3. 戚继光抗倭**

- 明朝戚继光组建"**戚家军**"抗倭，在台州**九战九捷**，平定浙东地区的倭患；随后又平息福建、广东地区的倭患，使东南沿海的倭患基本解除。
- 诗句："**封侯非我意，但愿海波平。**"

**4. 明朝的建筑艺术**

- **明代长城：** 东起**鸭绿江**，西至**嘉峪关**（比较：秦长城西起**临洮**，东到**辽东**）
- **紫禁城（故宫）：** 明朝修建，是当时世界上最宏大、最辉煌的皇家建筑群。

**5. 明朝的科技名著**

| 著作 | 作者 | 内容 |
| :--- | :--- | :--- |
| 《**本草纲目**》 | **李时珍** | 总结我国古代药物学成就，被誉为"**东方医学巨典**" |
| 《**天工开物**》 | **宋应星** | 被誉为"**中国17世纪的工艺百科全书**" |
| 《**农政全书**》 | **徐光启** | 明代末年重要的农业科学巨著 |

#### 二、清朝时期

**6. 清朝对边疆的管理（★★★高频考点）**

| 地区 | 措施 |
| :--- | :--- |
| **台湾** | 郑成功从荷兰殖民者手中收复台湾（1662年）；1684年，清设**台湾府**，隶属福建省；1885年，台湾正式建省 |
| **西藏** | 册封**达赖**、**班禅**；设**驻藏大臣** |
| **新疆** | 康熙皇帝平定**葛尔丹**叛乱；乾隆皇帝平定**大小和卓**叛乱，设**伊犁将军** |
| **蒙古** | **土尔扈特部落**在渥巴锡的领导下于1771年回归祖国，得到乾隆帝的妥善安置 |

**7. 清朝加强君主专制**

- **军机处：** 雍正时期设立，使君主专制进一步强化，**君主专制达到顶峰**。
- **文字狱：** 推行文字狱和文化专制政策，维护了集权统治，但**禁锢思想，阻碍社会发展**。

**8. 清代文学艺术**

| 类别 | 内容 |
| :--- | :--- |
| **京剧** | 乾隆时，**四大徽班**进京；道光年间形成新剧种"皮黄戏"，即京剧；京剧是中国文化的"**国粹**"之一 |
| **《红楼梦》** | 清代艺术成就最高、影响最深远的小说；**曹雪芹**创作前80回，高鹗续写后40回；深刻反映了封建社会末期的社会现实 |

**9. 清朝"闭关锁国"政策**

- **表现：** 颁布"禁海令"；1757年，关闭其他港口，只开放**广州**一处通商口岸，特许**广州十三行**经营对外贸易事务。
- **影响：**
  - 积极：对西方殖民者的侵略活动曾起到过一定的**自卫作用**。
  - 消极：导致国家的闭塞，使中国错失了向西方学习先进科学知识和生产技术的机会，中国**逐渐落伍于世界历史的发展进程**。

⚠️ **易错点：**
- 清朝乾隆平定的回部贵族**大小和卓**叛乱，他们是**新疆维吾尔族**的，不是回族。
- 郑成功收复台湾是从**荷兰**殖民者手中收复的，不是葡萄牙。
- 明代长城东起**鸭绿江**，秦长城东到**辽东**，注意区分。
- 军机处是**雍正**时期设立的，不是康熙或乾隆。

### Day 7（周日）：中国近代史开端（1840-1900）

**复习目标：** 掌握晚清三大不平等条约的内容及影响，理解近代化早期探索的异同。

#### 一、中国开始沦为半殖民地半封建社会

**1. 鸦片战争（1840-1842）（★★★大题考点）**

- **导火索：** 1839年**林则徐虎门销烟**（显示了中华民族反抗外来侵略的坚强意志）
- **结果：** 中英签订《**南京条约**》（中国近代史**第一个**不平等条约）

| 条款 | 内容 |
| :--- | :--- |
| 割地 | 割**香港岛**给英国 |
| 赔款 | 赔款**2100万**银元 |
| 通商 | 开放**广州、福州、厦门、宁波、上海**五处为通商口岸 |
| 关税 | **协议关税**（英商进出口货物应纳税款，必须经过双方协议） |

- **影响：** 中国从封建社会开始沦为**半殖民地半封建社会**，鸦片战争是**中国近代史的开端**。

**2. 第二次鸦片战争（1856-1860）**

- **英法联军火烧圆明园**（1860年）
- **沙俄**通过不平等条约共割占我国**150多万平方公里**领土
- **影响：** 中国的半殖民地化程度进一步加深

**3. 太平天国运动**

- 1851年1月，**洪秀全**在广西**金田村**发动武装起义，建号太平天国。
- 1853年3月，将南京改名为**天京**，作为都城。
- 1864年太平天国运动失败。
- **性质：** 中国历史上规模最宏大的一次**农民战争**。

**4. 甲午中日战争（1894-1895）（★★★大题考点）**

- **邓世昌：** 致远舰管带，黄海海战中壮烈牺牲。
- **结果：** 签订《**马关条约**》（1895年）

| 条款 | 内容 |
| :--- | :--- |
| 割地 | 割**辽东半岛**、**台湾全岛**及所有附属岛屿、**澎湖列岛**给日本 |
| 赔款 | 白银**2亿两** |
| 通商 | 增开沙市、重庆、苏州、杭州为商埠 |
| 设厂 | 允许日本在中国通商口岸**开设工厂** |

- **影响：** 大大加深了中国半殖民地化程度；列强掀起瓜分中国狂潮。

**5. 八国联军侵华（1900-1901）（★★★大题考点）**

- **原因：** 镇压**义和团**反帝爱国运动
- **结果：** 1901年签订《**辛丑条约**》

| 条款 | 内容 |
| :--- | :--- |
| 赔款 | 白银**4.5亿两** |
| 禁反 | 清政府保证严禁人民参加各种形式的**反帝活动** |
| 军事 | 拆毁大沽炮台；允许外国军队驻扎在从北京到山海关的铁路沿线要地 |
| 外交 | 划定北京**东交民巷**为使馆界；改总理衙门为**外务部**，班列六部之前 |

- **特点：** 赔款数目最庞大、主权丧失最严重的不平等条约。
- **影响：** 清政府沦为帝国主义统治中国的工具，中国**完全陷入半殖民地半封建社会**的深渊。

---

#### 二、近代化的早期探索

**6. 洋务运动（19世纪60至90年代）（★★★高频考点）**

- **口号：** 自强和求富
- **代表人物：** 中央**奕**，地方**曾国藩、李鸿章、左宗棠、张之洞**
- **军事工业：** 安庆内军械所（第一个）、江南制造总局（规模最大）、福州船政局
- **民用工业：** 轮船招商局、开平煤矿、汉阳铁厂、湖北织布局
- **地位：** 中国历史上**第一次近代化运动**，是**中国近代化的开端**
- **失败原因：** 根本目的是维护和巩固清政府的统治，再加上内部腐败和外国势力的挤压，**没有使中国走上富强的道路**。

**7. 戊戌变法（1898年）（★★★高频考点）**

- **代表人物：** **康有为、梁启超**等维新派
- **序幕：** **公车上书**
- **百日维新（1898年6月—9月）：** 光绪皇帝颁布"明定国是"诏书，实行变法

| 领域 | 内容 |
| :--- | :--- |
| 政治 | 裁撤冗官冗员，允许官民上书言事 |
| 经济 | 鼓励私人兴办工矿企业，发展农、工、商业 |
| 教育 | 废除**八股**，改试策论；开办新式学堂（**京师大学堂**，辛亥革命后改为北京大学） |
| 军事 | 裁减绿营，训练新式军队 |

- **影响：** 戊戌变法虽然失败，但在**思想文化方面**产生广泛而持久的影响。

**8. 近代化探索对比**

| 运动 | 时间 | 阶级 | 主张 | 结果 |
| :--- | :--- | :--- | :--- | :--- |
| 洋务运动 | 19世纪60-90年代 | 地主阶级 | 学习西方**技术** | 失败，但开启近代化 |
| 戊戌变法 | 1898年 | 资产阶级维新派 | 学习西方**政治制度**（君主立宪） | 失败，但思想影响深远 |
| 辛亥革命 | 1911年 | 资产阶级革命派 | 推翻封建帝制，建立**共和国** | 推翻清朝，但未改变半殖民地性质 |
| 新文化运动 | 1915年起 | 资产阶级激进派 | 学习西方**思想文化** | 思想解放，为五四运动奠基 |

⚠️ **易错点：**
- 中国近代史的开端是**1840年鸦片战争**，不是洋务运动。
- 《南京条约》割让的是**香港岛**给英国，不是割整个香港（香港包括香港岛、九龙、新界）。
- **火烧圆明园**是**英法联军**（1860年），不是八国联军。
- 八国联军侵华的主要目的是**镇压义和团运动**，根本目的是维护和扩大在华利益。
- 洋务运动是一次**封建统治者的自救运动**，不是资产阶级性质的改革，它只学习西方技术，没有触动封建制度。
- 与清政府签订《辛丑条约》的是**十一国**，不是八国。

### Day 8（周一）：新民主主义革命的开始与国共合作

**复习目标：** 掌握辛亥革命的意义，理解五四运动的地位，了解中国共产党的诞生与国共合作历程。

**对应教材：** 八年级上册 第三单元、第四单元、第五单元

---

#### 一、资产阶级民主革命与中华民国的建立

**1. 辛亥革命（★★★大题考点）**

- **孙中山：** 1905年成立**中国同盟会**（第一个全国规模的统一的资产阶级革命政党），以"**三民主义**"（民族主义、民权主义、民生主义）为指导思想；民主革命的先行者。
- **武昌起义：** 1911年**10月10日**爆发，为辛亥革命首义。
- **中华民国南京临时政府：** 1912年**1月1日**成立，孙中山任临时大总统；民国纪年法（公元X年-1911=民国Y年）。
- **《中华民国临时约法》：** 1912年3月颁布，中国历史上第一部**资产阶级共和国宪法**性质的重要文件。

**辛亥革命的意义：**
- 政治：推翻了清朝的反动统治，宣告了中国**两千多年君主专制制度**的终结；建立了中华民国。
- 思想：极大地推动了中华民族的**思想解放**，打开了中国进步潮流的大门。
- 经济：促成民族资本主义发展的"**短暂春天**"。
- 社会：促进了自由、平等新风尚的传播。

**2. 北洋政府与袁世凯复辟**

- 1912年3月，**袁世凯**在北京就任中华民国临时大总统，辛亥革命的胜利果实被袁世凯窃取。
- 袁世凯准备在1916年元旦举行登基大典，复辟帝制。孙中山发表《讨袁檄文》，**护国战争**爆发。1916年3月，袁世凯被迫宣布取消帝制。

---

#### 二、新民主主义革命的开始

**3. 新文化运动（1915年起）**

- **代表人物：** **陈独秀、胡适、鲁迅**
- **内容：** 抨击旧道德和旧文化；提倡**民主与科学**；文学革命——提倡**白话文**
- **口号：** **民主、科学**
- **性质：** 伟大的**思想解放运动**
- **作用：** 动摇了封建道德礼教的统治地位，使中国人民接受了一次民主与科学的洗礼，为随后爆发的**五四运动**起了思想宣传和铺垫的作用。

**4. 五四运动（1919年5月4日）（★★★大题考点）**

- **导火线：** **巴黎和会**中国外交的失败（中国提出废除"二十一条"等要求被拒绝，山东权益被转让给日本）
- **口号：** "**外争主权，内除国贼**"；"誓死力争，还我青岛"；"废除二十一条"
- **经过：** 斗争主力由**学生**变为**工人阶级**；运动中心由**北京**转移到**上海**
- **结局：** 北洋政府被迫释放被捕学生，罢免曹汝霖等人职务，拒绝在"巴黎和约"上签字，运动取得**初步胜利**
- **性质：** 是一场彻底**反帝反封建**的伟大爱国革命运动
- **意义：** 促进了马克思主义在中国的传播；促进了马克思主义同中国工人运动的结合；为中国共产党成立做了思想上和干部上的准备；是中国旧民主主义革命走向**新民主主义革命**的转折点

> 📝 **填空考点：** "五四精神"：崇尚民主与科学，追求进步，不屈不挠，敢于斗争的**爱国主义**精神。

**5. 中国共产党成立（★★★大题考点）**

- **诞生标志：** 中共**一大**的召开
- **时间：** 1921年7月
- **地点：** 上海（后移至浙江嘉兴**南湖**的游船上）
- **代表：** 毛泽东、董必武、李达等**13人**
- **内容：** 通过第一个党纲和奋斗目标（实现**共产主义**）；确定党的中心任务（领导工人运动）；选举中央领导机构（**陈独秀**为中央局书记）
- **意义：** 中共的诞生是**开天辟地**的大事，中国革命的面貌**焕然一新**。

---

#### 三、从国共合作到国共对立

**6. 第一次国共合作与北伐战争**

- **国民党一大（1924年1月）：** 标志着国共两党合作的正式建立。
- **黄埔军校（1924年5月）：** 蒋介石任校长，**周恩来**担任政治部主任；培养出大批军事和政治人才。
- **北伐战争（1926-1927年）：** 目的是推翻北洋军阀统治，统一全国；对象是**吴佩孚、孙传芳、张作霖**；重要战役：**汀泗桥战役**和**贺胜桥战役**；叶挺独立团的英勇善战为其所在的第四军获得"**铁军**"的称号。

**7. 南昌起义（1927年8月1日）——八一建军节**

- **领导者：** **周恩来、贺龙、朱德**等
- **意义：** 打响了武装反抗国民党反动统治的**第一枪**

**8. 井冈山会师**

- 1927年10月，毛泽东创建第一个农村革命根据地——**井冈山**革命根据地。
- 1928年，朱德、陈毅率部与毛泽东领导的工农红军在**井冈山**胜利会师，合编为中国工农红军第四军，**朱德**任军长，**毛泽东**任党代表。

**9. 红军长征（1934年10月至1936年10月）（★★★大题考点）**

- **根本原因：** "左"倾错误；直接原因：红军第五次反"围剿"失利
- **过程：** 从**瑞金**出发→冲破四道封锁线，渡过**湘江**（人员损失过半）→强渡**乌江**，改道贵州→**遵义会议**→四渡**赤水**→巧渡**金沙江**→强渡大渡河，飞夺**泸定桥**→翻雪山，过草地→1935年10月，党中央和红一方面军到达陕甘革命根据地，与陕北红军在**吴起镇**胜利会师→1936年10月，红军三大主力在**甘肃会宁**会师，宣告长征胜利结束
- **意义：** 粉碎了国民党反动派消灭红军的企图，保存了党和红军的基干力量，使中国革命**转危为安**。

**10. 遵义会议（1935年1月）（★★★高频考点）**

- **内容：** 纠正博古等人在军事上和组织上的"左"倾错误；肯定了毛泽东的正确军事主张；选举**毛泽东**为中央政治局常委；取消了博古、李德的军事最高指挥权。
- **意义：** 开始确立以**毛泽东**为代表的马克思主义的正确路线在中共中央的领导地位；在极其危急的时刻，挽救了党、挽救了红军、挽救了革命，是党的历史上**生死攸关的转折点**。这次会议是中国共产党从幼年走向**成熟**的标志。

⚠️ **易错点：**
- 辛亥革命推翻了清王朝的统治，结束了两千多年的封建君主专制制度，但**并未结束**中国半殖民地半封建社会的历史。
- 五四运动取得了**阶段性胜利**（或初步胜利），不是彻底胜利。
- 五四运动是中国新民主主义革命的开端，依据是**工人阶级**在斗争中以独立姿态登上政治舞台。
- 孙中山先生**没有亲身参与**武昌起义。
- 中共一大的地点是**上海**（后移至嘉兴南湖），不是北京。

### Day 9（周二）：抗日战争与解放战争

**复习目标：** 掌握抗日战争的主要史实和胜利原因，了解解放战争的主要战役。

**对应教材：** 八年级上册 第六单元、第七单元

---

#### 一、中华民族的抗日战争

**1. 九一八事变（1931年）**

- 1931年9月18日夜，日军制造**柳条湖事件**，进攻北大营，炮轰沈阳城。
- **九一八事变**成为中国人民抗日战争的起点，揭开了**世界反法西斯战争**的序幕。
- 4个多月后，**东北三省**沦陷。中国共产党派**杨靖宇**等人在东北组织游击队，中国人民的**局部抗战**开始了。

**2. 西安事变（1936年12月12日）（★★★高频考点）**

- **过程：** **张学良、杨虎城**扣押蒋介石，实行"兵谏"，通电全国，要求停止内战，联共抗日。中共派**周恩来**等人前往调停，蒋介石被迫接受停止内战、联共抗日的主张，西安事变**和平解决**。
- **意义：** 揭开了国共两党由内战到联合抗日的序幕，成为**扭转时局**的关键。从此，**十年内战**基本结束，**抗日民族统一战线**初步形成。

**3. 七七事变（1937年7月7日）（★★★高频考点）**

- 又称"**卢沟桥事变**"，日军以演习为借口进攻卢沟桥，炮轰宛平城。
- **标志着中国人民全民族抗战的开始。**
- 第二次国共合作：工农红军改编为**八路军、新四军**奔赴前线；1937年9月，以国共合作为主体的**抗日民族统一战线**正式建立。

**4. 南京大屠杀**

- 1937年12月13日，南京失陷，国民政府迁往**重庆**（战时陪都）。
- 日军占领南京后六周之内，屠杀手无寸铁的中国居民和放下武器的士兵达**30万**以上。

**5. 正面战场与敌后战场**

| 战役 | 时间 | 指挥者 | 意义 |
| :--- | :--- | :--- | :--- |
| **台儿庄战役** | 1938年3月 | 国民党**李宗仁** | 抗战以来中国正面战场取得的**最大**的一场胜仗 |
| **百团大战** | 1940年下半年 | 共产党八路军**彭德怀** | 有力打击了日军的侵略气焰，提高了共产党和八路军的威望 |

**6. 中共七大（1945年4月，延安）**

- 大会确定**毛泽东思想**为中国共产党的指导思想，并写入党章。
- 为争取抗日战争的最后胜利准备了条件，并为中国共产党和中国人民指明了战后的奋斗方向。

**7. 抗战胜利**

- 1945年**8月15日**，日本宣布无条件投降。
- **9月2日**，正式签署投降书，标志抗日战争的胜利。（9月3日为**抗日战争胜利纪念日**）

**抗战胜利的原因：**
1. 中国人民巨大的**民族觉醒**、空前的**民族团结**和英勇的**民族抗争**，是决定性因素。
2. 中国共产党在全民族团结抗战中发挥了**中流砥柱**的作用。
3. 世界上一切爱好和平与正义的国家和人民的支持。

**抗战胜利的意义：**
1. 中国抗日战争是中国近代以来反抗外敌入侵**第一次取得完全胜利**的民族解放战争。
2. 中国战场是世界反法西斯战争的**东方主战场**，对世界反法西斯战争的胜利作出了巨大贡献。

---

#### 二、人民解放战争

**8. 重庆谈判（1945年）**

- 蒋介石三次电邀毛泽东到重庆面商国家大事；毛泽东亲赴重庆，与国民党进行和平谈判。
- **结果：** 1945年10月10日，国共签订《政府与中共代表会谈纪要》，即"**双十协定**"。

**9. 刘邓大军挺进大别山（1947年夏）**

- **刘伯承、邓小平**率领晋冀鲁豫解放军主力，挺进大别山，揭开了人民解放军**战略进攻**的序幕。

**10. 三大战役（★★★大题考点）**

| 战役 | 时间 | 指挥者 | 意义 |
| :--- | :--- | :--- | :--- |
| **辽沈战役** | 1948年9月—11月 | 林彪、罗荣桓 | 歼灭国民党军47万余人，解放东北全境 |
| **淮海战役** | 1948年11月—1949年1月 | 刘伯承、陈毅、邓小平等 | 歼灭国民党军55万余人，解放长江中下游以北地区 |
| **平津战役** | 1948年11月—1949年1月 | 林彪、罗荣桓、聂荣臻 | 歼灭和改编国民党军52万余人，和平解放北平 |

- **影响：** 国民党主力基本被消灭，加速了人民解放战争在全国的胜利。

**11. 南京解放（渡江战役，1949年4月23日）**

- 人民解放军百万雄师横渡长江，解放南京，**结束了国民党在大陆的统治**。

**近代社会生活变迁（八上第八单元）**

- **张謇：** "状元实业家"，提出"**实业救国**"的口号，创办了大生纱厂等企业。
- **社会习俗变化：** 辛亥革命后，民国政府颁布了剪辫、易服和劝禁缠足等法令；废除跪拜礼，代之以鞠躬、握手礼。
- **教育发展：** 戊戌变法时期创办**京师大学堂**（中国近代第一所国立最高学府）；1905年，清朝**废除科举制度**。

⚠️ **易错点：**
- 抗日战争胜利的根本原因是**全民族团结抗战**，而不是某一单一因素。
- 三大战役的顺序：**辽沈战役→淮海战役→平津战役**。
- 中国拒绝在《凡尔赛合约》上签字的最主要原因是中国人民的反帝活动（五四运动）。
- 西安事变是**和平解决**的，不是武力解决。

### Day 10（周三）：中华人民共和国的成立与巩固

**复习目标：** 理解新中国成立的历史意义，掌握抗美援朝、土地改革、三大改造的内容。

**对应教材：** 八年级下册 第一单元、第二单元

---

#### 一、中华人民共和国的成立和巩固

**1. 政治协商会议（1949年9月）**

- 1949年9月，中国人民政治协商会议第一届全体会议在北平召开。
- **内容：** ①决定成立中华人民共和国；②通过了《中国人民政治协商会议**共同纲领**》（具有**临时宪法**作用）；③选举毛泽东为中央人民政府主席；④决定改北平为**北京**，作为新中国的首都；⑤以《**义勇军进行曲**》为代国歌；⑥以**五星红旗**为国旗；⑦决定建立**人民英雄纪念碑**。
- **意义：** 初步建立了中国共产党领导的**多党合作和政治协商**制度。

**2. 开国大典（1949年10月1日）（★★★大题考点）**

- 毛泽东向全世界宣告中华人民共和国中央人民政府成立。
- **意义：** 中华人民共和国的成立，开辟了中国历史的新纪元。中国人民经过一百多年的英勇斗争，终于推翻了帝国主义、封建主义和官僚资本主义的统治。中国真正成为**独立自主**的国家，占人类总数四分之一的中国人从此**站起来了**。是**中国现代史的开端**。

**3. 抗美援朝（1950-1953）（★★★高频考点）**

- **目的：** 抗美援朝，保家卫国
- **军队：** 中国人民志愿军，总司令：**彭德怀**
- **英雄人物：** **黄继光**（上甘岭战役，用身躯堵住敌人枪口）；**邱少云**（严守潜伏纪律，烈火中壮烈牺牲）
- **结果：** 1953年7月，美国被迫在《**朝鲜停战协定**》上签字，中朝人民取得反侵略战争的胜利。
- **意义：** 为新中国的建设赢得了一个相对稳定的和平环境，大大提高了我国的**国际地位**。

**4. 土地改革（1950-1952）（★★★高频考点）**

- **开始标志：** 1950年，中央人民政府颁布《**中华人民共和国土地改革法**》，废除地主阶级封建剥削的土地所有制，实行农民的土地所有制。
- **意义：** 彻底摧毁了**两千多年**的封建土地制度，消灭了地主阶级，农民成为土地的主人；大大解放了农村生产力，促进了农业发展；为国家**工业化建设**准备了条件。

---

#### 二、社会主义制度的建立与建设的探索

**5. 第一个五年计划（1953-1957年）**

- **重点：** 优先发展**重工业**，建立国家工业化和国防现代化的初步基础。
- **成就：** 鞍山钢铁公司、长春第一汽车制造厂、沈阳第一机床厂、武汉长江大桥等。

**6. 三大改造（1953-1956年底）（★★★大题考点）**

| 对象 | 改造途径 |
| :--- | :--- |
| 农业 | 走**合作化、集体化**道路（参加生产合作社） |
| 手工业 | 走**合作化、集体化**道路（参加生产合作社） |
| 资本主义工商业 | 通过**赎买政策**，实行**公私合营** |

- **意义：** 到1956年底，三大改造基本完成，标志着我国初步建立起**社会主义的基本制度**，我国进入**社会主义初级阶段**。

**7. 人民代表大会制度**

- 1954年9月，第一届全国人民代表大会制定了《**中华人民共和国宪法**》，这是我国**第一部社会主义类型**的宪法。
- 形成了**人民代表大会制度**，是我国的**根本政治制度**。

**8. 社会主义建设时期的模范人物**

- 大庆石油工人"铁人"**王进喜**；党的好干部**焦裕禄**；解放军的好战士**雷锋**。

**9. "文化大革命"（1966-1976年）**

- 形成**林彪、江青**两大反革命集团，民主与法制荡然无存。
- 国家主席**刘少奇**被迫害致死，是"文革"中最大的一起冤案。
- **结束标志：** 1976年**江青反革命集团**被粉碎。
- **危害：** 给党、国家和各族人民带来新中国成立后最严重的挫折，造成了巨大的损失。

⚠️ **易错点：**
- 结束两千多年**封建帝制**的是辛亥革命；结束两千多年**封建土地所有制**的是1952年土地改革的完成；结束**半殖民地半封建社会**历史的是开国大典（1949年）。
- 三大改造完成的标志是**1956年**，标志着社会主义制度在我国基本建立。
- 中国第一部社会主义类型的宪法是**1954年宪法**，不是1949年的《共同纲领》（《共同纲领》是临时宪法）。

### Day 11（周四）：中国特色社会主义道路

**复习目标：** 掌握十一届三中全会的历史意义，了解改革开放的主要内容和成就，理解"一国两制"与祖国统一。

#### 一、中国特色社会主义道路

**1. 十一届三中全会（1978年底）（★★★大题考点）**

- **内容：** ①思想上确立**解放思想、开动脑筋、实事求是、团结一致向前看**的指导方针；②政治上做出把党和国家的工作重心转移到**经济建设**上，实行**改革开放**的历史性决策；③组织上形成以**邓小平**为核心的中共中央领导集体。
- **意义：** 是新中国成立以来党的历史上具有深远意义的**伟大转折**，开启了我国改革开放历史新时期。

> 📝 **填空考点：** 为十一届三中全会召开奠定思想基础的是**关于真理标准问题的大讨论**（实践是检验真理的唯一标准）。

**2. 家庭联产承包责任制**

- 十一届三中全会以后，经济体制改革首先从**农村**开始。

**3. 经济特区（★★★高频考点）**

- **创建（1980年）：** 在**深圳（窗口城市）、珠海、汕头、厦门**设置经济特区。
- **发展：** 1988年增设**海南**经济特区；1990年**上海浦东**开发区建立。
- **格局：** 形成了"经济特区——沿海开放区域——沿海经济开放区——内地"的全方位、多层次、宽领域的对外开放格局。
- **作用：** 对引进**外资、先进技术和管理经验**，推动国内的进一步改革，扩大对外经济交流，发展社会主义现代化事业起到了极为重要的作用。

**4. 邓小平与改革开放**

**5. 社会主义市场经济体制**

- **提出：** 1992年，中国共产党**第十四次全国代表大会**明确提出要建立**社会主义市场经济体制**。
- **作用：** 把社会主义基本制度和市场经济结合起来，有利于实现经济的协调发展和稳定高速增长。

### Day 12（周五）：世界古代史

**复习目标：** 掌握古代文明的主要成就，理解封建制度的形成，了解亚洲封建国家的特点。

**对应教材：** 九年级上册 第一至四单元

---

#### 一、古代亚非文明

**1. 四大文明古国**

| 文明 | 地区 | 代表成就 |
| :--- | :--- | :--- |
| 古埃及 | 非洲**尼罗河**流域 | **金字塔**（法老陵墓，古埃及文明的象征）；象形文字 |
| 古巴比伦 | 西亚**两河**流域 | 《**汉谟拉比法典**》（迄今已知世界上第一部较为完整的**成文法典**） |
| 古印度 | **印度河、恒河**流域 | **种姓制度**（婆罗门、刹帝利、吠舍、首陀罗）；佛教 |
| 中国 | **黄河**流域 | 甲骨文、青铜器、丝绸 |

**2. 佛教**

- 创立于公元前6世纪的**古印度**，创始人**乔达摩·悉达多**（释迦牟尼）
- 提倡**众生平等**，是世界**三大宗教**之一

---

#### 二、古代欧洲文明

**3. 古希腊文明（★★★高频考点）**

- 古代希腊文明被称为**西方文明之源**。
- **雅典的民主政治：** 伯里克利当政期间，奴隶主民主政治发展到**高峰**；最高权力机构是**公民大会**，由全体**成年男性公民**组成（妇女、奴隶、外邦人除外）。

**4. 古罗马**

- 公元前27年，**屋大维**建立**罗马帝国**。
- 公元2世纪，罗马帝国进入黄金时期，版图横跨欧亚非三洲，地中海成为其内湖。
- **《十二铜表法》：** 是后世罗马法典乃至欧洲法学的渊源。
- **公历的缘起：** 凯撒命人以埃及太阳历为蓝本编制新的历法，称"**儒略历**"，是今天人们使用的公历的基础。

**5. 希腊罗马古典文化**

| 领域 | 内容 |
| :--- | :--- |
| 文学 | 《**荷马史诗**》是了解早期希腊社会的主要文献 |
| 雕塑 | 《**掷铁饼者**》是希腊雕塑艺术中的杰作 |
| 哲学 | **苏格拉底**（"人应该认识你自己"）；**亚里士多德**（创立逻辑学等新学科） |

---

#### 三、封建时代的欧洲

**6. 基督教**

- 创立于公元1世纪的**巴勒斯坦**地区，经典为《**圣经**》，宣扬耶稣为救世主。
- 公元4世纪被**罗马帝国**定为国教。现行**公元纪年**以及**圣诞节**与其有关。

**7. 封建制度**

- **法兰克王国：** 西罗马帝国灭亡前后，日耳曼人建立的最强大的王国，**克洛维**建立，他皈依了基督教。
- **封君与封臣：** "**我的附庸的附庸不是我的附庸**"；封君与封臣的关系有严格的等级性，且权力、义务交织，具**契约性**。
- **庄园：** 西欧中世纪社会的基础，是独立的**自给自足**的经济和政治单位；庄园法庭具有**司法权**。
- **西欧中世纪的城市：** 10世纪兴起；城市争取自由、自治的方式：**金钱赎买**和**武力斗争**（如法兰西的**琅城**）；随着城市的发展，形成**市民阶级**，富裕的大手工业作坊主、商人和银行家发展成为早期的**资产阶级**。

**8. 拜占庭帝国**

- 527年，**查士丁尼**组织编纂了《**查士丁尼法典**》《法学汇攥》《法理概要》《新法典》，统称为《**罗马民法大全**》，奠定了欧洲民法的基础。

---

#### 四、封建时代的亚洲国家

**9. 日本大化改新（★★★高频考点）**

- 7世纪中期，日本**孝德天皇**仿照**隋唐制度**进行改革，史称"**大化改新**"。
- **内容：** 政治上建立以**天皇**为中心的中央集权制度；经济上将贵族的土地收归国有，部民转为国家公民，国家定期把土地分给公民耕种，统一收取赋税。
- **意义：** 大化改新使日本发展成为一个**中央集权制的封建国家**。

**10. 伊斯兰教与阿拉伯帝国**

- **伊斯兰教：** 创立于公元7世纪的阿拉伯半岛的**麦加**，创立者为**穆罕默德**，信徒称为**穆斯林**。
- **阿拉伯文化的贡献：** 改造了古代印度人从0到9的计算法，形成了"**阿拉伯数字**"；创造了完整的**代数学**；编著了《医学集成》和《**医典**》；《**天方夜谭**》是阿拉伯文学的瑰宝。
- **阿拉伯人的作用：** 担当了**沟通东西方文化**的角色；中国的造纸术、指南针、火药等重大发明和印度的棉花、食糖等，都由阿拉伯人传入欧洲。

⚠️ **易错点：**
- 雅典民主政治的参与者是**成年男性公民**，妇女、奴隶、外邦人**不能**参与政治，因此雅典民主政治是**少数人**的民主。
- 《汉谟拉比法典》是世界上第一部较为完整的**成文法典**，不是第一部法典。
- 大化改新是仿照**隋唐**制度进行的，不是仿照秦汉制度。

### Day 13（周六）：世界近现代史

**复习目标：** 掌握文艺复兴、新航路开辟、资产阶级革命、工业革命的主要内容，了解两次世界大战和战后格局。

**对应教材：** 九年级上册 第五至七单元、九年级下册 全册

#### 一、步入近代

**1. 文艺复兴（★★★高频考点）**

- **时间：** 14-17世纪
- **兴起地：** **意大利**，后传播于欧洲
- **核心思想：** **人文主义**（以人为中心，反对神学束缚）
- **实质：** 一场**资产阶级思想解放运动**（对古希腊罗马文化有继承更有创新）

| 代表人物 | 国家 | 代表作 |
| :--- | :--- | :--- |
| **但丁** | 意大利 | 《**神曲**》（文艺复兴的先驱） |
| **达·芬奇** | 意大利 | 《**蒙娜丽莎**》《**最后的晚餐**》 |
| **莎士比亚** | 英国 | 《**罗密欧与朱丽叶**》《**哈姆雷特**》 |

- **影响：** 促进了人们思想的大解放，推动了欧洲思想文化领域的繁荣，为欧洲资本主义的产生奠定了**思想文化基础**。

**2. 新航路开辟（★★★高频考点）**

| 航海家 | 国籍 | 时间 | 成就 |
| :--- | :--- | :--- | :--- |
| **迪亚士** | 葡萄牙 | 1488年 | 到达**好望角** |
| **达·伽马** | 葡萄牙 | 1497-1498年 | 到达**印度**（首先实现西欧人到达东方的基本目标） |
| **哥伦布** | 意大利（西班牙资助） | 1492年 | 横渡大西洋，到达今天的**美洲** |
| **麦哲伦** | 葡萄牙（西班牙资助） | 1519-1522年 | 率船队完成**环球航行** |

- **意义：** 使大西洋沿岸的工商业经济繁荣起来；世界开始**连成一个整体**；促进了资本主义的产生和发展。

**3. 三角贸易**

- 欧洲殖民者从非洲掳获黑奴卖往美洲充当劳动力，这种以**贩卖黑奴**为中心的贸易被称为"**三角贸易**"。
- **航程：** ①出程：欧洲→非洲（掳掠黑人）；②中程：非洲→美洲（贩卖黑奴）；③归程：美洲→欧洲（满载金银和工业原料）。
- **影响：** 促进了欧洲的**资本原始积累**，有助于世界市场逐渐形成，但也给殖民地人民带来深重灾难。

#### 二、资本主义制度的初步确立

**4. 英美法资产阶级革命对比（★★★大题考点）**

| 项目 | 英国资产阶级革命 | 美国独立战争 | 法国大革命 |
| :--- | :--- | :--- | :--- |
| 时间 | 1640-1688年 | 1775-1783年 | 1789-1794年 |
| 导火线/起因 | 议会与国王矛盾 | 北美殖民地反英 | 财政危机，三级会议召开 |
| 重要文件 | 《**权利法案**》（1689年） | 《**独立宣言**》（1776年）；1787年宪法 | 《**人权宣言**》（1789年） |
| 政体 | **君主立宪制** | **联邦共和制** | 共和制（后拿破仑帝制） |
| 结束标志 | 1688年**光荣革命** | 1783年英国承认美国独立 | 1794年热月政变 |

**5. 拿破仑帝国**

- 拿破仑于1804年加冕称帝，建立**法兰西第一帝国**。
- **对内：** 颁布《**拿破仑法典**》，确立资本主义社会的立法规范，有利于法国资本主义的发展。
- **对外：** 多次打败反法同盟，捍卫法国大革命的成果；传播法国革命的思想，客观上有利于欧洲封建制度的瓦解。但后期演变为对外侵略的战争，最终走向覆亡。

#### 三、工业革命和工人运动

**6. 第一次工业革命（★★★大题考点）**

- **时间：** 18世纪60年代，英国开始
- **重要发明：**
  - 1765年，**哈格里夫斯**发明纺纱机"**珍妮机**"
  - **瓦特**改进蒸汽机，首先在纺织部门投入使用；到19世纪30年代，蒸汽机成为主要动力来源
  - 1825年，英国人**斯蒂芬森**发明蒸汽机车（火车）正式试车，标志着**铁路时代**的来临
- **意义：** 工业革命极大提高了社会生产力水平，从此人类进入"**蒸汽时代**"。19世纪中期，英国成为第一个**工业国家**。

**7. 马克思主义诞生**

- 1848年，**马克思、恩格斯**共同起草的《**共产党宣言**》发表，标志着**马克思主义的诞生**。
- 从此，无产阶级的斗争有了科学理论的指导，社会主义运动更加蓬勃地发展起来。

#### 四、殖民地人民的反抗与资本主义制度的扩展

**8. 殖民地人民的反抗**

- **拉丁美洲独立运动：** **玻利瓦尔**和**圣马丁**领导拉丁美洲人民反抗**西班牙**和**葡萄牙**殖民统治，被誉为南美的"**解放者**"。
- **印度民族大起义（1857-1859年）：** 印度土兵和印度王公反抗英国殖民者，**章西女王**壮烈牺牲。

**9. 俄国1861年改革**

- 俄国沙皇**亚历山大二世**颁布废除农奴制法令：农奴获得**人身自由**，同时可以获得一份土地，但必须**出钱赎买**，所出的价钱高于当时的地价。
- **意义：** 俄国历史上一个重要转折点，推动俄国走上**发展资本主义**的道路。但保留了大量封建残余。

**10. 美国内战（1861-1865年）**

- 1862年，林肯领导的联邦政府颁布《**解放黑人奴隶宣言**》，宣布从1863年元旦起，南方叛乱地区的奴隶永远获得自由。
- **性质：** 美国历史上**第二次资产阶级革命**，维护了国家统一，废除了奴隶制度，扫清了资本主义发展的又一大障碍。

**11. 日本明治维新（★★★高频考点）**

| 领域 | 内容 |
| :--- | :--- |
| 政治 | **废藩置县**，加强中央集权 |
| 军事 | 实行**征兵制**，建立新式军队 |
| 经济 | "**殖产兴业**"，引进西方先进技术，大力发展近代资本主义工商业 |
| 文化 | 提倡"**文明开化**"，向欧美学习，改造日本的教育、文化、生活方式 |

- **影响：** 是日本历史的重大转折点，通过明治维新，使日本迅速走上了资本主义发展道路，实现了富国强兵，开始跻身资本主义强国之列。但保留了大量旧制度残余，军国主义色彩浓厚，日本很快走上了**对外侵略扩张**的道路。

#### 五、第二次工业革命和近代科学文化

**12. 第二次工业革命（★★★高频考点）**

- **时间：** 19世纪六七十年代开始
- **特点：** **科学研究同工业生产紧密结合**
- **重要成就：**
  - 电力成为新的能源，人类进入"**电气时代**"；1879年，美国"发明大王"**爱迪生**研制成功耐用白炽灯泡
  - 19世纪80年代，德国人**卡尔·本茨**制造出内燃机驱动的汽车；1913年，美国企业家**福特**使用流水线生产汽车
  - 1903年，美国**莱特兄弟**制成飞机并试飞成功
- **影响：** 促进了生产力的发展，资本主义由自由资本主义向**垄断资本主义（帝国主义）**阶段过渡。

**13. 近代科学**

- **牛顿（英国）：** 近代自然科学的奠基人之一，著有《**自然哲学的数学原理**》；三大成就：**万有引力定律**、光学分析、微积分学。
- **达尔文（英国）：** 出版了《**物种起源**》，提出了**进化论**的观点，是生物科学的一次伟大革命。

#### 六、两次世界大战

**14. 第一次世界大战（1914-1918）（★★★高频考点）**

- **两大集团：** "三国同盟"（**德国、意大利、奥匈帝国**）；"三国协约"（**英国、法国、俄国**）
- **导火线：** **萨拉热窝事件**（1914年6月28日，奥匈帝国皇储斐迪南大公夫妇被刺杀）
- **主要战役：** **凡尔登战役**（1916年，德、法，被称为"**绞肉机**"）
- **性质：** 一场非正义的**帝国主义战争**

**15. 俄国十月革命（1917年）**

- 1917年由**列宁**领导，推翻资产阶级临时政府，史称"十月革命"（彼得格勒武装起义）。
- **意义：** 人类第一次取胜的**社会主义革命**，世界第一个**社会主义国家**诞生。

**16. 凡尔赛—华盛顿体系**

- **巴黎和会：** 协约国与德国签订《**凡尔赛和约**》，由美英法三国操纵，是"一战"战胜的帝国主义列强**瓜分世界的一次分赃会议**。
- **华盛顿会议：** 签订《**九国公约**》，要求中国"门户开放，机会均等"，维持了几个帝国主义国家**共同支配中国**的局面。
- **评价：** "凡尔赛—华盛顿体系"是战后帝国主义重新瓜分世界的体系，不可能消除帝国主义国家之间的矛盾，因此不可能长期维持下去。

**17. 苏俄-苏联的社会主义建设**

- **新经济政策（1921年，列宁）：** 以**征收粮食税**代替余粮征集制；允许私人经营中小企业；实行自由贸易；实行按劳取酬的工资制。影响：调动了生产者的积极性，迅速缓解了危机，促进国民经济稳步发展。
- **斯大林模式（1936年）：** 1928-1937年，苏联先后完成两个五年计划，由落后的农业国变成了强大的工业国；1936年新宪法通过，标志着**苏联体制（斯大林模式）**的形成；特征：**高度集中的计划经济体制**和**高度集权的政治体制**。

**18. 第二次世界大战（1939-1945）（★★★大题考点）**

| 阶段 | 时间 | 事件 |
| :--- | :--- | :--- |
| 开始 | 1939年 | 德国进攻**波兰**，标志"二战"全面爆发 |
| 扩大 | 1941年6月 | 德国进攻苏联，苏德战场爆发 |
| 扩大 | 1941年12月 | 日本偷袭美国**珍珠港**，太平洋战场爆发 |
| 联合 | 1942年1月1日 | 美英苏中等26个国家签订《**联合国家宣言**》，**世界反法西斯联盟**建立 |
| 转折 | 1942-1943年 | 苏联取得**斯大林格勒战役**的胜利，是苏德战场和"二战"的转折点 |
| 胜利 | 1944年6月 | **诺曼底登陆**，开辟欧洲第二战场 |
| 胜利 | 1945年 | 美英苏召开**雅尔塔会议**；苏军发动柏林战役，德国投降；日本签署投降书（9月2日），二战全面结束 |

#### 七、冷战和战后世界格局

**19. 冷战（★★★高频考点）**

- **开始标志：** 1947年3月，**杜鲁门主义**的出台，标志着美苏战时同盟关系正式破裂，美苏之间的"冷战"开始。
- **具体表现：** 政治上：杜鲁门主义；经济上：**马歇尔计划**；军事上：建立**北大西洋公约组织**和**华约**（1955年苏联）。
- **两极格局形成：** 华沙条约组织的成立，美苏两极格局形成。
- **冷战结束：** 1991年苏联解体，两极格局结束。

**20. 欧洲的联合**

- **欧共体（1967年）：** 欧洲煤钢共同体、欧洲经济共同体和欧洲原子能共同体3个组织合并。
- **欧盟（1993年）：** 前身是欧共体，大大加快了欧洲一体化的进程；欧盟成员国使用统一的货币是**欧元**。

**21. 苏联改革与解体**

- **赫鲁晓夫改革（1953年）：** 批判斯大林个人崇拜，进行经济改革；但并未从根本上改变苏联高度集中的经济政治体制，1964年赫鲁晓夫被迫下台。
- **戈尔巴乔夫改革（1985年）：** 首先把经济体制作为改革重点，但没有取得成果；又把改革重点转到政治改革上，取消苏共领导地位，实行**多党制**；使人们思想混乱，局势迅速失控。
- **苏联解体（1991年12月）：** 直接原因：戈尔巴乔夫改革失败；1991年发生"**八一九事件**"，成为加速苏联解体的催化剂。

**22. 亚非拉国家的新发展**

- **万隆会议（1955年）：** 亚洲、非洲29个国家和地区的代表召开，发展中国家作为一支新兴独立的政治力量登上了国际舞台；中国提出的"**求同存异**"方针受到国际社会的认可。
- **"非洲年"（1960年）：** 非洲17个国家获得独立，这一年因此被称为"非洲年"。1990年，**纳米比亚**独立，标志着所有非洲国家都摆脱了殖民主义的枷锁。
- **古巴：** 卡斯特罗等人推翻了美国支持的独裁政权，走上了**社会主义**发展道路。
- **巴拿马：** 1999年，巴拿马从美国手中收回了**运河区**的全部主权。

**23. 联合国与世界贸易组织**

- **联合国：** 1945年10月成立，首要宗旨是**维持国际和平及安全**。
- **世界贸易组织（WTO）：** 1995年1月建立，宗旨是以非歧视性、开放、公平为原则，促进全球贸易和经济发展。

⚠️ **易错点：**
- 文艺复兴的实质是**资产阶级思想解放运动**，而不是古代希腊罗马文化的简单复兴。
- 首先直接实现西欧人开辟新航路基本目标（到达东方）的是**达·伽马**，而不是哥伦布（哥伦布到达的是美洲）。
- 英国资产阶级革命**结束**的标志是1688年的"**光荣革命**"，而不是《权利法案》的颁布（后者标志着君主立宪制的确立）。
- 俄国1861年改革和日本明治维新都是**自上而下**的资产阶级性质的改革，都保留了大量的**封建残余**。
- 第一次世界大战的**导火线**是萨拉热窝事件，**根本原因**是帝国主义国家政治经济发展不平衡。
- 罗斯福新政**没有改变**资本主义的本质，也**无法从根本上消除**经济危机。
- 二战中，**斯大林格勒战役**是二战的重要转折点，而不是莫斯科保卫战。

### 近5年高频考点汇总（补充考点标注）

以下为近5年北京中考历史中**反复出现**的考点，需重点掌握：

**【补充考点1】北京地区历史（北京特色考点）**
- 北京人（周口店）遗址——世界文化遗产
- 北京地区的历史沿革：蓟城→幽州→燕京→大都→北京
- 元大都的建设（忽必烈，1267年开始建设）
- 明代北京城的建设（永乐年间迁都北京，修建紫禁城）
- 北京作为首都的历史意义

**【补充考点2】宋代市舶司**
- 宋代在广州、泉州、明州等地设置**市舶司**，专门管理海外贸易事务，征收商税。
- 这是中国历史上最早的专门管理对外贸易的机构之一。

**【补充考点3】一二九运动（1935年）**
- 1935年12月9日，北平学生举行抗日救国示威游行，史称"一二九运动"。
- 意义：揭露了日本帝国主义侵略中国、吞并华北的阴谋，打击了国民党政府的妥协投降政策，促进了全国抗日救亡运动新高潮的到来。

**【补充考点4】中国人民政治协商会议**
- 1949年9月召开，通过《共同纲领》，具有**临时宪法**的作用。
- 1954年第一届全国人民代表大会召开后，政协不再代行全国人大职能，但作为**多党合作和政治协商**的机构继续存在。

**【补充考点5】启蒙运动**
- 17-18世纪兴起于欧洲（主要在法国），是继文艺复兴之后的又一次伟大的思想解放运动。
- 核心思想：**理性主义**
- 代表人物：**伏尔泰**（法国，反对专制）、**孟德斯鸠**（法国，三权分立）、**卢梭**（法国，社会契约论）
- 影响：为美国独立战争、法国大革命提供了思想武器；推动了欧美资产阶级革命的发展。
