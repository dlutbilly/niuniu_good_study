# 07 历史

**考试类型：** 等级考查科目 | **成绩呈现：** A/B/C/D 四等级 | **命题：** 各区自主命题

> 注：历史不计入中考总分，但等级成绩影响校额到校录取资格（要求 A 等）。C 等及以上为合格，D 等为不合格。

---

## 考试内容框架

| 历史模块 | 主要考查内容 |
|----------|--------------|
| 中国古代史 | 先秦、秦汉、三国两晋南北朝、隋唐、宋元、明清 |
| 中国近代史 | 鸦片战争至新中国成立（1840-1949） |
| 中国现代史 | 新中国成立至今 |
| 世界古代史 | 古代文明、封建社会 |
| 世界近代史 | 文艺复兴、工业革命、资产阶级革命 |
| 世界现代史 | 两次世界大战、冷战、当代世界 |

## 快速提分重点（冲击 A 等）

1. **时间线**：建立清晰的历史时间轴，避免时序混乱
2. **因果关系**：掌握重大历史事件的背景、经过、影响
3. **横向比较**：中外历史对比，提升分析能力
4. **关键词记忆**：历史意义和影响的标准表述需熟记
