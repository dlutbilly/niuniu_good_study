# 12 信息科技

**考试类型：** 其他考查科目 | **成绩呈现：** 等级 | **考核方式：** 过程性评价

---

## 考查内容说明

信息科技学科主要考查学生的数字素养与信息技术应用能力，包括计算机基础操作、网络应用、编程思维等内容，以过程性评价为主。

---

## 目录内容规划

| 文件名 | 内容说明 |

| `01_计算机基础.md` | 计算机硬件软件基础知识 |
| `02_网络与信息安全.md` | 互联网基础、信息安全意识 |
| `03_编程思维基础.md` | 算法基础、程序设计入门 |
| `04_数字素养.md` | 信息检索、数据处理基础 |

---

## 备考建议

信息科技以过程性评价为主，建议：
1. 认真参与日常信息科技课程实践操作
2. 掌握基本的计算机操作技能
3. 了解信息安全相关知识
