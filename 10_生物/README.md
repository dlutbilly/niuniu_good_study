# 10 生物

**考试类型：** 等级考查科目 | **成绩呈现：** A/B/C/D 四等级 | **命题：** 各区自主命题

> 注：生物不计入中考总分，但等级成绩影响校额到校录取资格（要求 A 等）。C 等及以上为合格，D 等为不合格。

---

## 考试内容框架

| 生物模块 | 主要考查内容 |
|----------|--------------|
| 生命的物质基础 | 细胞结构、生命活动的物质基础 |
| 生物体的结构层次 | 细胞、组织、器官、系统 |
| 生物与环境 | 生态系统、生物圈 |
| 生物圈中的绿色植物 | 光合作用、呼吸作用、蒸腾作用 |
| 人体生命活动 | 消化、呼吸、循环、排泄、神经调节 |
| 遗传与进化 | 遗传变异、进化理论 |
| 生物技术 | 发酵技术、转基因技术 |

## 快速提分重点（冲击 A 等）

1. **光合作用与呼吸作用**：两大核心生理过程，必须深刻理解
2. **人体系统**：各系统的结构与功能是高频考点
3. **遗传规律**：基因与性状的关系、遗传图解
4. **实验分析**：生物实验的设计原理与结果分析
