# 09 化学

**考试类型：** 等级考查科目 | **成绩呈现：** A/B/C/D 四等级 | **命题：** 各区自主命题

> 注：化学不计入中考总分，但等级成绩影响校额到校录取资格（要求 A 等）。C 等及以上为合格，D 等为不合格。

---

## 考试内容框架

| 化学模块 | 主要考查内容 |
|----------|--------------|
| 化学基础概念 | 物质分类、化学变化、元素与化合物 |
| 化学实验 | 实验操作、实验现象分析 |
| 物质结构 | 原子结构、化学键、离子 |
| 化学反应 | 四大基本反应类型、氧化还原 |
| 酸碱盐 | 酸碱盐的性质与转化 |
| 化学计算 | 化学方程式计算、溶液计算 |
