# 北京中考化学考点梳理与14天复习计划

## 第一部分：考情分析与复习策略

### 1. 北京中考化学命题趋势（参考2024年真题）
北京市初中学业水平考试化学试题依据义务教育化学课程标准，坚持以学定考、面向全体、回归教材。试卷结构通常包含单项选择题（约25题）和非选择题（生活现象解释、科普阅读理解、生产实际分析、基本实验及其原理分析、科学探究、实际应用定量计算等题型）。

**核心命题特点：**
- **强基固本：** 重点考查物质的变化与分类、化学用语（化学式、化学方程式）、金属材料、水与溶液、酸碱盐等主干知识。
- **素养导向：** 强调宏观与微观相结合、定性与定量相结合的思维能力（如微观示意图分析、溶质质量分数计算）。
- **情境真实：** 试题素材多源于现行教材、课堂教学及国家时事要闻（如嫦娥六号月背采样、能源绿色转型等），并渗透中华优秀传统文化（如老北京“九门走九车”、二十四节气）。
- **重视实验：** 考查基本实验操作、气体制备与性质探究，并在形式上不断创新（如通过连线建立仪器与名称的联系，降低书写难度）。

### 2. 高效复习策略建议
- **构建知识网络：** 化学知识点繁杂，需通过思维导图将元素化合物知识、基本概念、化学反应规律串联起来。
- **攻克高频易错点：** 重点关注化学式书写、化合价计算、实验操作细节（如先撤导管还是先熄灭酒精灯）及物质检验除杂。
- **精练典型真题：** 针对北京中考特色题型（如科普阅读、生产实际分析、科学探究），总结解题模型和答题模板。
- **回归课本实验：** 熟练掌握教材中的重点实验原理、装置图、现象及结论。

## 第三部分：14天复习计划每日核心考点与易错点梳理

### Day 1：走进化学世界与我们周围的空气
**复习目标：** 区分物理/化学变化，掌握常见仪器操作，熟记空气成分及氧气的性质与制备。

**核心知识点：**
1. **物质的变化与性质：**
   - 物理变化：没有生成新物质的变化（如水结冰、玻璃破碎、干冰升华）。
   - 化学变化：生成新物质的变化（如燃烧、铁生锈、食物腐烂）。
   - 物理性质：颜色、状态、气味、熔点、沸点、密度、溶解性、挥发性等。
   - 化学性质：可燃性、助燃性、还原性、氧化性、酸碱性、毒性等。
2. **化学实验基本操作：**
   - 药品的取用：固体（镊子或药匙，一横二放三慢竖）、液体（倾倒法：瓶塞倒放，标签向手心）。
   - 物质的加热：用外焰加热，试管内液体不超过容积的1/3。
   - 仪器的洗涤：玻璃仪器洗净标准是内壁附着的水既不聚成水滴，也不成股流下。
3. **空气的成分与保护：**
   - 体积分数：氮气（78%）、氧气（21%）、稀有气体（0.94%）、二氧化碳（0.03%）。
   - 空气污染：有害气体（二氧化硫、二氧化氮、一氧化碳）和烟尘。
4. **氧气的性质：**
   - 物理性质：无色无味气体，密度比空气大，不易溶于水。
   - 化学性质：具有助燃性，能与多种物质反应（如木炭、硫、铁丝）。
5. **氧气的制备：**
   - 实验室制法：加热高锰酸钾（或氯酸钾）、分解过氧化氢。
   - 工业制法：分离液态空气（物理变化）。

⚠️ **易错点标记：**
- 错把“发光发热”作为判断化学变化的唯一标准（电灯发光发热是物理变化）。
- 混淆“烟”（固体小颗粒）和“雾”（小液滴），如红磷燃烧产生大量白烟，而不是白雾。
- 铁丝在氧气中燃烧实验时，集气瓶底未放少量水或细沙，导致瓶底炸裂。
- 实验室制取氧气（加热固体制气体并用排水法收集）结束时，先熄灭酒精灯，后将导管移出水面，导致水倒吸引起试管破裂。

### Day 2：物质构成的奥秘与自然界的水
**复习目标：** 掌握微观粒子的概念，理解元素与化合价，熟悉水的净化方法及电解水实验。

**核心知识点：**
1. **微观粒子（分子、原子、离子）：**
   - 分子：保持物质化学性质的最小粒子。
   - 原子：化学变化中的最小粒子。
   - 离子：带电的原子或原子团。
2. **元素与元素周期表：**
   - 元素：具有相同质子数（即核电荷数）的一类原子的总称。
   - 元素符号的意义：表示一种元素，表示该元素的一个原子。
   - 元素周期表：横行叫周期，纵行叫族。原子序数 = 核电荷数 = 质子数 = 核外电子数。
3. **化合价与化学式：**
   - 化合价规律：单质中元素化合价为零；化合物中正负化合价代数和为零。
   - 常见化合价口诀：一价氢氯钾钠银，二价氧钙钡镁锌，三铝四硅五氮磷。
   - 化学式的意义：宏观（表示一种物质及组成元素）、微观（表示一个分子及构成）。
4. **水的净化与水资源保护：**
   - 净化方法：沉淀、过滤、吸附（活性炭去色去味）、蒸馏（净化程度最高）。
   - 硬水与软水：硬水含较多可溶性钙镁化合物，用肥皂水区分（硬水泡沫少起浮渣），煮沸或蒸馏可降低硬度。
5. **水的组成（电解水实验）：**
   - 现象：正极产生氧气（能使带火星木条复燃），负极产生氢气（能燃烧，产生淡蓝色火焰），体积比约 1:2（正氧负氢，氢二氧一）。
   - 结论：水是由氢元素和氧元素组成的。

⚠️ **易错点标记：**
- 混淆宏观与微观概念：如说“水是由两个氢原子和一个氧原子组成的”（应为“水分子”）。
- 误认为过滤能降低水的硬度（过滤只能除去不溶性杂质，不能除去可溶性钙镁离子）。
- 计算相对分子质量时，漏乘元素符号右下角的数字。

### Day 3：化学方程式与质量守恒定律
**复习目标：** 深入理解质量守恒定律，熟练书写化学方程式并进行相关计算。

**核心知识点：**
1. **质量守恒定律：**
   - 内容：参加化学反应的各物质的质量总和，等于反应后生成的各物质的质量总和。
   - 微观解释（六个不变）：反应前后原子的种类、数目、质量不变；元素的种类、质量不变；物质的总质量不变。
   - 两个一定改变：物质的种类、分子的种类一定改变。
   - 一个可能改变：分子的总数可能改变。
2. **化学方程式的书写：**
   - 原则：以客观事实为基础，遵守质量守恒定律。
   - 步骤：写（反应物和生成物化学式）→配（配平，使两边各原子数目相等）→注（注明反应条件及生成物状态“↑”或“↓”）。
3. **化学反应的基本类型：**
   - 化合反应：多变一（A + B → AB）。
   - 分解反应：一变多（AB → A + B）。
   - 置换反应：单质+化合物 → 新单质+新化合物（A + BC → AC + B）。
   - 复分解反应：两种化合物互相交换成分生成另外两种化合物（AB + CD → AD + CB）。
4. **根据化学方程式的计算：**
   - 步骤：设未知量 → 写出化学方程式 → 写出相关物质的相对分子质量和已知量、未知量 → 列比例式求解 → 简明写出答案。

⚠️ **易错点标记：**
- 解释质量守恒定律时，忽略“参加反应”的前提（未反应完的物质质量不能计入）。
- 验证质量守恒定律实验时，若有气体参与或生成，未在密闭容器中进行。
- 书写化学方程式时，忘记配平或漏写反应条件、气体/沉淀符号。
- 计算时，相对分子质量计算错误，或未考虑化学式前面的化学计量数。

### Day 4：碳和碳的氧化物
**复习目标：** 掌握碳单质的物理性质差异及化学性质，熟记一氧化碳和二氧化碳的性质、制备与用途。

**核心知识点：**
1. **碳单质的性质与用途：**
   - 金刚石与石墨：物理性质差异大是因为碳原子的排列方式不同。金刚石硬度大（切割玻璃），石墨质软导电（铅笔芯、电极）。
   - C60：由分子构成，形似足球。
   - 化学性质：常温下稳定（古代字画保存长久）；可燃性（充分燃烧生成二氧化碳，不充分生成一氧化碳）；还原性（还原氧化铜生成铜和二氧化碳）。
2. **二氧化碳的制备：**
   - 实验室制法：大理石（或石灰石）与稀盐酸反应。
   - 原理：CaCO3 + 2HCl = CaCl2 + H2O + CO2↑
   - 收集方法：向上排空气法（密度比空气大，能溶于水）。
   - 检验：通入澄清石灰水，变浑浊。
   - 验满：燃着的木条放在集气瓶口，熄灭。
3. **二氧化碳的性质与用途：**
   - 物理性质：无色无味气体，密度比空气大，能溶于水。固体叫干冰，升华吸热（人工降雨）。
   - 化学性质：不能燃烧也不支持燃烧；与水反应生成碳酸（使紫色石蕊变红）；与澄清石灰水反应生成碳酸钙沉淀。
   - 用途：灭火、温室肥料、化工原料。
4. **一氧化碳的性质与用途：**
   - 物理性质：无色无味气体，难溶于水，密度比空气略小。
   - 化学性质：可燃性（产生蓝色火焰）；毒性（与血红蛋白结合）；还原性（冶炼金属）。

⚠️ **易错点标记：**
- 混淆碳单质物理性质差异的原因（是原子排列方式不同，而非原子种类不同）。
- 实验室制取二氧化碳时，错用稀硫酸（生成微溶物硫酸钙覆盖在石灰石表面阻止反应）或浓盐酸（挥发出的氯化氢使二氧化碳不纯）。
- 误认为二氧化碳有毒（二氧化碳无毒，只是不能供给呼吸，浓度高会导致窒息）。
- 一氧化碳还原氧化铜实验时，未进行尾气处理（点燃或收集），导致一氧化碳污染空气。

### Day 5：燃料及其利用
**复习目标：** 理解燃烧的条件与灭火的原理，掌握三大化石燃料及新能源的开发利用。

**核心知识点：**
1. **燃烧与灭火：**
   - 燃烧的条件（缺一不可）：可燃物、与氧气（或空气）接触、温度达到着火点。
   - 灭火的原理（破坏其一即可）：清除可燃物、隔绝氧气、降温至着火点以下。
   - 常见灭火器：水基型（扑灭木材纸张等固体火灾）、干粉（扑灭油、气火灾）、二氧化碳（扑灭图书档案、精密仪器火灾）。
2. **燃料的合理利用与开发：**
   - 三大化石燃料（不可再生）：煤（工业的粮食）、石油（工业的血液）、天然气（主要成分甲烷CH4）。
   - 化学反应中的能量变化：放热反应（如燃烧、生石灰与水反应）、吸热反应（如碳与二氧化碳反应）。
   - 化石燃料的燃烧对环境的影响：酸雨（二氧化硫、二氧化氮引起）、温室效应（二氧化碳引起）。
3. **新能源与绿色化学：**
   - 新能源：太阳能、风能、地热能、潮汐能、生物质能、核能。
   - 氢能源：最清洁的燃料（燃烧产物只有水），但制取成本高、贮存困难。
   - 绿色化学：从源头上消除污染，实现零排放。

⚠️ **易错点标记：**
- 灭火原理表述不严谨，如说“降低着火点”（着火点是物质的固有属性，不能降低，只能“降温至着火点以下”）。
- 混淆煤气中毒的原因（是一氧化碳中毒，而非煤气本身有毒）。
- 误认为所有爆炸都是化学变化（轮胎爆胎是物理变化，火药爆炸是化学变化）。

### Day 6：金属和金属材料
**复习目标：** 熟记金属活动性顺序及其应用，掌握铁的冶炼原理及金属锈蚀的防护。

**核心知识点：**
1. **金属材料：**
   - 纯金属：铁、铜、铝等，具有金属光泽、导电性、导热性、延展性。
   - 合金：在金属中加热熔合某些金属或非金属形成的具有金属特性的物质（如生铁和钢是含碳量不同的铁合金）。合金的硬度一般比纯金属大，熔点一般比纯金属低，抗腐蚀性更强。
2. **金属的化学性质：**
   - 与氧气反应：镁铝常温下能反应（表面形成致密氧化膜）；铁铜在高温下反应；金在高温下也不反应（真金不怕火炼）。
   - 与酸反应：排在氢前面的金属能置换出酸中的氢气。
   - 与盐溶液反应：排在前面的金属能把排在后面的金属从其盐溶液中置换出来。
3. **金属活动性顺序及其应用：**
   - 顺序：K Ca Na Mg Al, Zn Fe Sn Pb (H), Cu Hg Ag Pt Au。
   - 应用：判断金属与酸、金属与盐溶液能否发生置换反应；设计实验探究金属活动性强弱。
4. **铁的冶炼：**
   - 原理：在高温下，用一氧化碳把铁从铁矿石里还原出来（3CO + Fe2O3 =高温= 2Fe + 3CO2）。
   - 设备：高炉。
   - 原料：铁矿石、焦炭（提供热量、生成一氧化碳）、石灰石（除去杂质二氧化硅）。
5. **金属资源的保护：**
   - 铁生锈条件：与氧气和水同时接触。
   - 防锈措施：保持表面洁净干燥；涂保护层（刷漆、涂油、电镀）；制成合金（不锈钢）。
   - 保护途径：防止金属腐蚀、回收利用废旧金属、有计划合理开采矿物、寻找金属替代品。

⚠️ **易错点标记：**
- 误认为生铁和钢是纯净物（它们都是铁合金，是混合物）。
- 描述铁丝在氧气中燃烧的产物时，错写为氧化铁（应为四氧化三铁）。
- 铁与酸或盐溶液发生置换反应时，生成的是亚铁盐（Fe2+），溶液呈浅绿色。
- 一氧化碳还原氧化铁实验时，未先通一氧化碳排尽空气就加热，导致发生爆炸。

### Day 7：第一周综合自测与查漏补缺
**复习目标：** 回顾前六天核心考点，通过一套阶段模拟卷检验复习效果，针对错题进行专项突破。

### Day 8：溶液（一）：组成与特征
**复习目标：** 理解溶液的概念与特征，掌握溶解度曲线的应用。

**核心知识点：**
1. **溶液的形成：**
   - 概念：一种或几种物质分散到另一种物质里，形成均一的、稳定的混合物。
   - 特征：均一性、稳定性、混合物。
   - 溶质与溶剂：溶质可以是固、液、气；水是最常用的溶剂（酒精、汽油也可作溶剂）。
   - 溶解时的吸放热：氢氧化钠固体、浓硫酸溶于水放热；硝酸铵固体溶于水吸热；氯化钠溶于水温度基本不变。
2. **饱和溶液与不饱和溶液：**
   - 概念：在一定温度下，向一定量溶剂里加入某种溶质，当溶质不能继续溶解时，所得的溶液叫做饱和溶液。
   - 相互转化：一般情况下，增加溶质、蒸发溶剂、降低温度可使不饱和溶液变为饱和溶液（氢氧化钙除外，降温使其溶解度增大）。
3. **溶解度与溶解度曲线：**
   - 概念：在一定温度下，某固态物质在100g溶剂里达到饱和状态时所溶解的质量。
   - 四要素：一定温度、100g溶剂、达到饱和状态、单位是克（g）。
   - 溶解度曲线应用：查某温度下的溶解度；比较不同物质在同一温度下的溶解度大小；判断结晶方法（陡升型用降温结晶，缓升型用蒸发结晶）。

⚠️ **易错点标记：**
- 误认为溶液都是无色的（如硫酸铜溶液是蓝色）。
- 判断饱和溶液时，未指明“一定温度”和“一定量溶剂”。
- 混淆饱和溶液与浓溶液的关系（饱和溶液不一定是浓溶液，同温同溶质的饱和溶液比不饱和溶液浓）。

### Day 9：溶液（二）：浓度与计算
**复习目标：** 熟练掌握溶质质量分数的计算，掌握一定溶质质量分数溶液的配制。

**核心知识点：**
1. **溶质质量分数：**
   - 概念：溶质质量与溶液质量之比。
   - 公式：溶质质量分数 = 溶质质量 / 溶液质量 × 100%。
   - 饱和溶液中溶质质量分数 = 溶解度 / (100g + 溶解度) × 100%。
2. **溶液的稀释与浓缩计算：**
   - 原理：稀释前后溶质的质量不变。
   - 公式：浓溶液质量 × 浓溶液溶质质量分数 = 稀溶液质量 × 稀溶液溶质质量分数。
3. **一定溶质质量分数溶液的配制：**
   - 步骤：计算、称量（或量取）、溶解、装瓶贴标签。
   - 仪器：托盘天平、量筒、烧杯、玻璃棒、药匙、胶头滴管。
   - 误差分析：若溶质质量偏小（如称量时物码放反且使用游码、固体有杂质），或溶剂质量偏大（如仰视读数、烧杯内壁有水），则质量分数偏小。

⚠️ **易错点标记：**
- 计算溶质质量分数时，将未溶解的固体质量计入溶质质量。
- 配制溶液时，量筒读数方法错误（仰视导致实际体积偏大，俯视导致实际体积偏小）。
- 混淆溶质质量分数与溶解度的概念（前者是比值，后者是具体质量，受温度影响）。

### Day 10：酸和碱（一）：常见酸碱性质
**复习目标：** 掌握常见酸碱的物理性质及用途，熟记酸碱的化学性质。

**核心知识点：**
1. **常见的酸（盐酸、硫酸）：**
   - 物理性质：浓盐酸有挥发性（瓶口有白雾）；浓硫酸有吸水性（作干燥剂）、脱水性、强氧化性。
   - 化学性质：能使紫色石蕊变红，无色酚酞不变色；与活泼金属反应生成氢气；与金属氧化物反应（除锈）；与碱发生中和反应；与某些盐反应生成新酸和新盐。
   - 浓硫酸的稀释：酸入水，沿器壁，慢慢倒，不断搅（切不可将水倒入浓硫酸中）。
2. **常见的碱（氢氧化钠、氢氧化钙）：**
   - 物理性质：氢氧化钠（俗称火碱、烧碱、苛性钠），极易溶于水放热，易潮解（作干燥剂）；氢氧化钙（俗称熟石灰、消石灰），微溶于水，水溶液叫澄清石灰水。
   - 化学性质：能使紫色石蕊变蓝，无色酚酞变红；与非金属氧化物反应（如吸收二氧化碳）；与酸发生中和反应；与某些盐反应生成新碱和新盐。
3. **酸碱的通性原因：**
   - 酸溶液中都含有氢离子（H+）。
   - 碱溶液中都含有氢氧根离子（OH-）。

⚠️ **易错点标记：**
- 浓硫酸稀释操作颠倒，导致酸液飞溅伤人。
- 误认为能使指示剂变色的都是酸或碱（如碳酸钠溶液呈碱性，能使酚酞变红，但它是盐）。
- 氢氧化钠敞口放置变质的原因（吸收空气中的水蒸气潮解，再与二氧化碳反应生成碳酸钠）。

### Day 11：酸和碱（二）：中和反应与pH
**复习目标：** 理解中和反应的实质及其应用，掌握溶液酸碱度（pH）的测定与意义。

**核心知识点：**
1. **中和反应：**
   - 概念：酸与碱作用生成盐和水的反应。
   - 实质：H+ + OH- = H2O。
   - 应用：改良酸性土壤（加熟石灰）；处理工厂废水（如用熟石灰中和硫酸厂废水）；医药应用（如用氢氧化铝治疗胃酸过多，被蚊虫叮咬涂抹肥皂水等碱性物质减痛）。
2. **溶液的酸碱度（pH）：**
   - 范围：0 ~ 14。
   - 规律：pH < 7 呈酸性（越小酸性越强）；pH = 7 呈中性；pH > 7 呈碱性（越大碱性越强）。
   - 测定方法：用玻璃棒蘸取待测液滴在pH试纸上，把试纸显示的颜色与标准比色卡比较，读出pH值（只能是整数）。
3. **pH的应用与环境意义：**
   - 正常雨水的pH约为5.6（溶有二氧化碳）；酸雨的pH小于5.6（溶有二氧化硫、二氧化氮）。
   - 农作物一般适宜在pH为6.5~7.5的土壤中生长。

⚠️ **易错点标记：**
- 测定pH时，先用水润湿pH试纸（相当于稀释待测液，导致酸性溶液pH偏大，碱性溶液pH偏小）。
- 误认为生成盐和水的反应都是中和反应（如金属氧化物与酸反应也生成盐和水，但不是中和反应）。
- 混淆酸碱性与酸碱度（酸碱性用指示剂测定，酸碱度用pH试纸测定具体强弱）。

### Day 12：盐化肥与化学与生活
**复习目标：** 掌握常见盐的性质及复分解反应条件，了解化学肥料及六大营养素。

**核心知识点：**
1. **常见的盐：**
   - 氯化钠（NaCl）：食盐的主要成分，用于调味、防腐、融雪剂、配制生理盐水。
   - 碳酸钠（Na2CO3）：俗称纯碱、苏打，用于玻璃、造纸、纺织、洗涤剂。
   - 碳酸氢钠（NaHCO3）：俗称小苏打，用于焙制糕点、治疗胃酸过多。
   - 碳酸钙（CaCO3）：石灰石、大理石的主要成分，用于建筑材料、补钙剂。
2. **复分解反应发生的条件：**
   - 反应物：若反应物中无酸，则必须均可溶于水。
   - 生成物：必须有沉淀、气体或水生成（满足其一即可）。
3. **化学肥料：**
   - 氮肥（N）：促进植物茎叶生长茂盛，叶色浓绿（缺氮叶片发黄）。
   - 磷肥（P）：促进作物根系发达，增强抗寒抗旱能力（缺磷植株矮小）。
   - 钾肥（K）：促进作物生长健壮，茎秆粗硬，增强抗病虫害和抗倒伏能力（缺钾叶片边缘发黄）。
   - 复合肥：含两种或两种以上营养元素（如KNO3、NH4H2PO4）。
4. **化学与生活（六大营养素）：**
   - 蛋白质（肉、蛋、奶、豆）、糖类（米、面、薯类）、油脂（植物油、动物脂肪）、维生素（蔬菜、水果）、无机盐、水。
   - 化学元素与人体健康：缺铁贫血；缺钙骨质疏松或佝偻病；缺碘甲状腺肿大；缺锌或锌过量导致甲状腺肿大；缺氟或氟过量产生龋齿或氟斑牙。
   - 有毒物质：甲醛（浸泡水产品防腐有毒）、黄曲霉素（霉变食物）、一氧化碳。

⚠️ **易错点标记：**
- 误认为纯碱是碱（纯碱是碳酸钠，属于盐，但其水溶液呈碱性）。
- 判断复分解反应能否发生时，未检查生成物是否满足条件。
- 铵态氮肥（如硝酸铵）与碱性物质（如熟石灰）混合施用，会放出氨气降低肥效。

### Day 13：重点实验与科学探究专项突破
**复习目标：** 梳理气体制备、物质检验除杂等重点实验，掌握探究题解题思路。

**核心知识点：**
1. **气体的制备与收集装置选择：**
   - 发生装置：依据反应物状态和反应条件（固固加热型、固液常温型）。
   - 收集装置：依据气体的密度和溶解性（排水法、向上/向下排空气法）。
2. **物质的检验与鉴别：**
   - 原则：操作简便、现象明显。
   - 常见离子的检验：H+（石蕊或活泼金属）、OH-（酚酞或铜盐溶液）、CO32-（加酸产生气体，通入澄清石灰水变浑浊）、Cl-（加硝酸银和稀硝酸产生白色沉淀）、SO42-（加氯化钡和稀硝酸产生白色沉淀）。
3. **物质的分离与除杂：**
   - 原则：不增（不引入新杂质）、不减（不减少被提纯物质）、易分（杂质转化为沉淀、气体或水易分离）。
   - 常见方法：物理方法（过滤、结晶）、化学方法（沉淀法、气化法、置换法）。
4. **科学探究题解题思路：**
   - 明确探究目的（提出问题）。
   - 分析猜想与假设的合理性（基于物质性质和反应原理）。
   - 设计实验方案（控制变量法、对比实验）。
   - 观察现象得出结论，进行反思与评价。

⚠️ **易错点标记：**
- 气体制备装置气密性检查方法描述不完整（如未说明“松开手后导管内形成一段水柱”）。
- 除杂时加入的试剂过量未处理，导致引入新杂质。
- 探究反应后溶液中溶质成分时，只考虑生成物，忽略了反应物可能过量。

## 附录：高频化学方程式速记

**一、化合反应（多变一）**
1. 木炭充分燃烧：C + O2 =点燃= CO2
2. 木炭不充分燃烧：2C + O2 =点燃= 2CO
3. 铁丝在氧气中燃烧：3Fe + 2O2 =点燃= Fe3O4
4. 镁条在空气中燃烧：2Mg + O2 =点燃= 2MgO
5. 氢气在空气中燃烧：2H2 + O2 =点燃= 2H2O
6. 一氧化碳燃烧：2CO + O2 =点燃= 2CO2
7. 生石灰与水反应：CaO + H2O = Ca(OH)2

**二、分解反应（一变多）**
8. 加热高锰酸钾：2KMnO4 =加热= K2MnO4 + MnO2 + O2↑
9. 分解过氧化氢：2H2O2 =MnO2= 2H2O + O2↑
10. 电解水：2H2O =通电= 2H2↑ + O2↑
11. 碳酸不稳定分解：H2CO3 = H2O + CO2↑
12. 高温煅烧石灰石：CaCO3 =高温= CaO + CO2↑

**三、置换反应（单换单）**
13. 锌与稀硫酸：Zn + H2SO4 = ZnSO4 + H2↑
14. 铁与稀盐酸：Fe + 2HCl = FeCl2 + H2↑
15. 铁与硫酸铜溶液：Fe + CuSO4 = FeSO4 + Cu
16. 碳还原氧化铜：C + 2CuO =高温= 2Cu + CO2↑

**四、复分解反应（双交换）**
17. 盐酸除铁锈：Fe2O3 + 6HCl = 2FeCl3 + 3H2O
18. 氢氧化钠与盐酸中和：NaOH + HCl = NaCl + H2O
19. 氢氧化钙与稀硫酸中和：Ca(OH)2 + H2SO4 = CaSO4 + 2H2O
20. 大理石与稀盐酸制二氧化碳：CaCO3 + 2HCl = CaCl2 + H2O + CO2↑
21. 碳酸钠与稀盐酸：Na2CO3 + 2HCl = 2NaCl + H2O + CO2↑
22. 检验硫酸根离子：BaCl2 + H2SO4 = BaSO4↓ + 2HCl

**五、其他重要反应**
23. 一氧化碳还原氧化铁：3CO + Fe2O3 =高温= 2Fe + 3CO2
24. 二氧化碳通入澄清石灰水：CO2 + Ca(OH)2 = CaCO3↓ + H2O
25. 氢氧化钠吸收二氧化碳：2NaOH + CO2 = Na2CO3 + H2O
26. 甲烷燃烧：CH4 + 2O2 =点燃= CO2 + 2H2O
27. 酒精燃烧：C2H5OH + 3O2 =点燃= 2CO2 + 3H2O
