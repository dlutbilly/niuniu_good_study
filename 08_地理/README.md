# 08 地理

**考试类型：** 等级考查科目 | **成绩呈现：** A/B/C/D 四等级 | **命题：** 各区自主命题

> 注：地理不计入中考总分，但等级成绩影响校额到校录取资格（要求 A 等）。C 等及以上为合格，D 等为不合格。

---

## 考试内容框架

| 地理模块 | 主要考查内容 |
|----------|--------------|
| 地球与地图 | 经纬度、地图判读、地球运动 |
| 世界地理 | 大洲大洋、气候、自然带、重要国家 |
| 中国地理 | 地形、气候、河流、区域地理 |
| 北京地理 | 北京自然与人文地理特征 |

## 快速提分重点（冲击 A 等）

1. **地图判读**：地图是地理学习的核心工具，必须熟练掌握
2. **气候类型**：各气候类型的分布规律和特征是必考内容
3. **中国地理**：重点掌握四大地理区域的自然和人文特征
4. **联系实际**：结合时事热点（如区域发展、环境问题）分析地理现象
